<SectionHeader n={1} title="Problem Overview" />

A phrase is a **palindrome** if, after converting all uppercase letters to lowercase and removing all non-alphanumeric characters, it reads the same forward and backward. Alphanumeric characters include letters and digits.

Given a string `s`, return `true` if it is a palindrome, or `false` otherwise.

**Constraints:**
- `1 <= s.length <= 2 * 10^5`
- `s` consists only of printable ASCII characters.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input: s = "A man, a plan, a canal: Panama"
Output: true
Explanation: "amanaplanacanalpanama" is a palindrome.
```

#### Example 2

```
Input: s = "race a car"
Output: false
Explanation: "raceacar" is not a palindrome.
```

#### Example 3

```
Input: s = " "
Output: true
Explanation: After removing non-alphanumeric characters, s is empty "". An empty string reads the same forward and backward.
```

<SectionHeader n={4} title="Approach" />

Convert the string to a character array. Use `left` starting at index 0 and `right` starting at the last index. Before comparing, independently advance each pointer past any character that is neither a letter (`a`–`z`, ASCII 97–122) nor a digit (`0`–`9`, ASCII 48–57). If both are alphanumeric and match (lowercased), advance both. If they do not match, return `false`.

<Callout type="cyan" label="ASCII-based check">
Checking ASCII ranges directly avoids calling any library character-classification functions. Lowercase with <code>lowercased()</code> first, then check if the ASCII value falls in 97–122 or 48–57.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Convert to character array">
    `let array = Array(s)` for O(1) index access.
  </StepItem>
  <StepItem title="Initialise two pointers">
    `var left = 0`, `var right = array.count - 1`.
  </StepItem>
  <StepItem title="Lowercase and check alphanumeric">
    For each pointer, lowercase the character: `let a = Character(array[left].lowercased())`. Check if its ASCII value is in `[97,122]` (a–z) or `[48,57]` (0–9). If not, skip (advance the pointer) and `continue`.
  </StepItem>
  <StepItem title="Compare">
    If both are alphanumeric: if they are equal, advance both (`left += 1`, `right -= 1`). Otherwise return `false`.
  </StepItem>
  <StepItem title="Return true">
    Once pointers cross or meet, all pairs have been verified — return `true`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func isPalindrome(_ s: String) -> Bool {
    let array = Array(s)
    var left  = 0
    var right = array.count - 1

    while left < right {
      let a = Character(array[left].lowercased())
      let b = Character(array[right].lowercased())

      let asciiA = a.asciiValue ?? 0
      if !((asciiA >= 97 && asciiA <= 122) || (asciiA >= 48 && asciiA <= 57)) {
        left += 1
        continue
      }

      let asciiB = b.asciiValue ?? 0
      if !((asciiB >= 97 && asciiB <= 122) || (asciiB >= 48 && asciiB <= 57)) {
        right -= 1
        continue
      }

      if a == b {
        left  += 1
        right -= 1
      } else {
        return false
      }
    }

    return true
  }
}
```
