<SectionHeader n={1} title="Problem Overview" />

Given an integer array `nums`, rotate the array to the **right** by `k` steps, where `k` is non-negative.

Rotating right by 1 step moves every element one position to the right, and the last element wraps around to the front.

**Constraints:**
- `1 <= nums.length <= 10^5`
- `-2^31 <= nums[i] <= 2^31 - 1`
- `0 <= k <= 10^5`
- The rotation must be done **in-place** with O(1) extra space.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input: nums = [1,2,3,4,5,6,7], k = 3
Output: [5,6,7,1,2,3,4]
Explanation: Rotate right by 3:
  [7,1,2,3,4,5,6] (k=1)
  [6,7,1,2,3,4,5] (k=2)
  [5,6,7,1,2,3,4] (k=3)
```

#### Example 2

```
Input: nums = [-1,-100,3,99], k = 2
Output: [3,99,-1,-100]
```

<SectionHeader n={4} title="Approach" />

Let `c = k % nums.count` (effective rotation). Then:
1. Reverse the entire array.
2. Reverse the first `c` elements.
3. Reverse the remaining `nums.count - c` elements.

<Callout type="cyan" label="Why three reverses work">
After reversing all, the last k elements (which should move to the front) are now at the front but in reverse order. Reversing the first k fixes their order. Reversing the rest fixes the second half. Together, the array is exactly the k-step right rotation.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Compute effective rotation">
    `let c = k % nums.count`. If `c == 0`, no work needed (return immediately).
  </StepItem>
  <StepItem title="Reverse entire array">
    Two-pointer swap from `l=0` to `r=nums.count-1`, swapping and converging.
  </StepItem>
  <StepItem title="Reverse first c elements">
    Two-pointer swap from `l=0` to `r=c-1`.
  </StepItem>
  <StepItem title="Reverse remaining elements">
    Two-pointer swap from `l=c` to `r=nums.count-1`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func rotate(_ nums: inout [Int], _ k: Int) {
    let c = k % nums.count
    guard c > 0 else { return }

    func reverse(_ l: Int, _ r: Int) {
      var l = l, r = r
      while l < r {
        nums.swapAt(l, r)
        l += 1; r -= 1
      }
    }

    reverse(0, nums.count - 1)  // Reverse all
    reverse(0, c - 1)           // Reverse first c
    reverse(c, nums.count - 1)  // Reverse rest
  }
}
```
