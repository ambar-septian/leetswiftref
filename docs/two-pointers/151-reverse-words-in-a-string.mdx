<SectionHeader n={1} title="Problem Overview" />

Given an input string `s`, reverse the order of the **words**.

A **word** is defined as a sequence of non-space characters. The words in `s` will be separated by at least one space.

Return a string of the words in reverse order concatenated by a single space.

**Note:** `s` may contain leading or trailing spaces or multiple spaces between two words. The returned string should have only a single space separating the words. Do not include any extra spaces.

**Constraints:**
- `1 <= s.length <= 10^4`
- `s` contains English letters (upper-case and lower-case), digits, and spaces.
- There is at least one word in `s`.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  s = "the sky is blue"
Output: "blue is sky the"
```

#### Example 2

```
Input:  s = "  hello world  "
Output: "world hello"
Explanation: Leading and trailing spaces are stripped; single space between words.
```

#### Example 3

```
Input:  s = "a good   example"
Output: "example good a"
Explanation: Multiple spaces between words are reduced to a single space.
```

<SectionHeader n={4} title="Approach" />

Walk the string with index `i`. Skip all spaces to find the start of a word, then advance `j` until the next space or end of string. Collect `s[i..<j]` as a `Substring`. After all words are collected, reverse the array and join with a single space.

<Callout type="cyan" label="Using Substring for efficiency">
Swift's <code>Substring</code> is a view into the original string — no copying until the final <code>joined</code>. This makes the scan allocation-free until the output is built.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Initialise collector and index">
    `var words: [Substring] = []`, `var i = s.startIndex`.
  </StepItem>
  <StepItem title="Outer loop: skip leading spaces">
    While `i less than s.endIndex` and `s[i] == " "`, advance `i`.
  </StepItem>
  <StepItem title="Inner loop: find word end">
    Let `j = i`. While `j less than s.endIndex` and `s[j] != " "`, advance `j`.
  </StepItem>
  <StepItem title="Collect word">
    `words.append(s[i..<j])`. Set `i = j` and repeat the outer loop.
  </StepItem>
  <StepItem title="Return reversed and joined">
    `words.reversed().joined(separator: " ")`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func reverseWords(_ s: String) -> String {
    var words: [Substring] = []
    var i = s.startIndex

    while i < s.endIndex {
      // Skip spaces
      while i < s.endIndex, s[i] == " " {
        i = s.index(after: i)
      }
      if i == s.endIndex { break }

      // Find word end
      var j = i
      while j < s.endIndex, s[j] != " " {
        j = s.index(after: j)
      }

      words.append(s[i..<j])
      i = j
    }

    return words.reversed().joined(separator: " ")
  }
}
```
