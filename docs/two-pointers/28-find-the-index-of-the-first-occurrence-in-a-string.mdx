<SectionHeader n={1} title="Problem Overview" />

Given two strings `haystack` and `needle`, return the index of the **first occurrence** of `needle` in `haystack`, or `-1` if `needle` is not part of `haystack`.

**Constraints**
- `1 <= haystack.length, needle.length <= 10^4`
- `haystack` and `needle` consist of only lowercase English characters.

<Callout type="green" label="Core Insight">
Anchor a left pointer at each starting position in haystack. Use a right pointer to match characters of needle one by one from that anchor. On full match return left; on any mismatch reset right to 0 and advance left by one.
</Callout>

<SectionHeader n={2} title="Examples" />

#### Example 1
```
Input:  haystack = "sadbutsad", needle = "sad"
Output: 0
Explanation: "sad" occurs at index 0 and 6. Return 0 (first occurrence).
```

#### Example 2
```
Input:  haystack = "leetcode", needle = "leeto"
Output: -1
Explanation: "leeto" does not occur in "leetcode".
```

<SectionHeader n={4} title="Approach" />

Convert both strings to character arrays for O(1) access. Maintain `left` as the current starting index in haystack and `right` as how many characters of needle have been matched so far. At each step, compare `h[left + right]` with `n[right]`. On match advance `right` and check if `right == n.count` (full match — return `left`). On mismatch reset `right = 0` and advance `left`. Return `-1` if left exhausts the haystack.

<Callout type="cyan" label="The left + right index trick">
Instead of copying a substring, index into haystack directly with left + right. This avoids O(m) allocations per window and keeps space O(1).
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Convert to arrays and guard">
    `h = haystack.map`, `n = needle.map`. Return -1 if either is empty.
  </StepItem>
  <StepItem title="Initialise pointers">
    `left = 0`, `right = 0`, `count = 0`.
  </StepItem>
  <StepItem title="Bounds check before access">
    Guard `right + left < h.count && right < n.count` — if either overflows, no match remains; return -1.
  </StepItem>
  <StepItem title="Match or reset">
    If `h[right + left] == n[right]`: advance `right`, increment `count`. If `count == n.count` return `left`. Else reset `count = 0`, `right = 0`, advance `left`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func strStr(_ haystack: String, _ needle: String) -> Int {
    let h = haystack.map { String($0) }
    let n = needle.map { String($0) }
    guard h.count > 0, n.count > 0 else { return -1 }

    var left = 0
    var right = 0
    var count = 0

    while left < h.count {
      guard right + left < h.count, right < n.count else { return -1 }

      if h[right + left] == n[right] {
        right += 1
        count += 1
        if count == n.count { return left }
        
      } else {
        count = 0
        left += 1
        right = 0
      }
    }

    return -1
  }
}
```
