<SectionHeader n={1} title="Problem Overview" />

The **Fibonacci numbers**, commonly denoted `F(n)`, form a sequence called the Fibonacci sequence, such that each number is the sum of the two preceding ones, starting from `0` and `1`. That is:

- `F(0) = 0`
- `F(1) = 1`
- `F(n) = F(n - 1) + F(n - 2)` for `n > 1`

Given `n`, calculate `F(n)`.

**Constraints:**
- `0 <= n <= 30`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  n = 2
Output: 1
Explanation: F(2) = F(1) + F(0) = 1 + 0 = 1
```

#### Example 2

```
Input:  n = 3
Output: 2
Explanation: F(3) = F(2) + F(1) = 1 + 1 = 2
```

#### Example 3

```
Input:  n = 4
Output: 3
Explanation: F(4) = F(3) + F(2) = 2 + 1 = 3
```

<SectionHeader n={4} title="Approach" />

Use a dictionary `memo` to cache results. For `n == 0` return `0`; for `n <= 2` return `1` (both base cases). For all other `n`, check the cache first; if missing, compute `fib(n-1) + fib(n-2)`, store in `memo[n]`, and return.

<Callout type="cyan" label="Why memoization over pure recursion">
Without memoization, F(5) calls F(4) and F(3); F(4) calls F(3) again. The call tree has exponential size (O(2^n)). With memoization each value is computed exactly once, giving O(n) time and O(n) space.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Handle base cases">
    If `n == 0`, return `0`. If `n less than or equal to 2`, return `1`.
  </StepItem>
  <StepItem title="Check the memo cache">
    If `memo[n]` exists, return it immediately.
  </StepItem>
  <StepItem title="Recurse and cache">
    Compute `result = fib(n - 1) + fib(n - 2)`. Store `memo[n] = result`.
  </StepItem>
  <StepItem title="Return result">
    Return `result`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  var memo = [Int: Int]()

  func fib(_ n: Int) -> Int {
    if n == 0 { return 0 }
    if n <= 2  { return 1 }

    if let result = memo[n] {
      return result
    }

    let result = fib(n - 1) + fib(n - 2)
    memo[n] = result
    return result
  }
}
```
