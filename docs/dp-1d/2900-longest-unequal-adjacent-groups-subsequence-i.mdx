<SectionHeader n={1} title="Problem Overview" />

You are given a string array `words` and a **binary** array `groups`, both of length `n`, where `groups[i]` is either `0` or `1`.

You need to select the **longest alternating subsequence** from `words`. A subsequence is alternating if consecutive selected elements come from different groups (i.e., `groups[i] != groups[j]` for any two consecutive selected indices `i` and `j`).

Return any valid longest alternating subsequence.

**Constraints:**
- `1 <= n <= 10`
- `1 <= words[i].length <= 10`
- `groups.length == n`
- `groups[i]` is either `0` or `1`.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  words = ["e","a","b"], groups = [0,0,1]
Output: ["e","b"]  (or ["a","b"])
Explanation: Select indices 0 and 2: groups differ (0,1). Length 2 is maximum.
```

#### Example 2

```
Input:  words = ["a","b","c","d"], groups = [1,0,1,1]
Output: ["a","b","c"]
Explanation: Select indices 0,1,2: groups (1,0,1) are alternating.
```

<SectionHeader n={4} title="Approach" />

Track the previously selected group (`prev`). Initialise it to `nil`. Iterate `words` with their `groups`. If `groups[i] != prev`, append `words[i]` to the result and update `prev = groups[i]`.

<Callout type="green" label="Why greedy is optimal for binary groups">
With only two group values, any time we skip an eligible word we either miss it entirely or pick a later word with the same group — which cannot extend the alternating run. Every eligible word should be taken immediately.
</Callout>

<StepList>

  </StepItem>
  <StepItem title="Iterate">
    For each `i` in `0..<words.count`: if `prev == nil || groups[i] != prev!`, append `words[i]` to result and set `prev = groups[i]`.
  </StepItem>
  <StepItem title="Return result">

  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func getLongestSubsequence(_ words: [String], _ groups: [Int]) -> [String] {
    var prev: Int? = nil
    var result = [String]()

    for i in 0..<words.count {
      if prev == nil || groups[i] != prev! {
        result.append(words[i])
        prev = groups[i]
      }
    }

    return result
  }
}
```
