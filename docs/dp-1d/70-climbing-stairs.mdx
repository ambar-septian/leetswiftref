<SectionHeader n={1} title="Problem Overview" />

You are climbing a staircase. It takes `n` steps to reach the top. Each time you can climb either `1` or `2` steps. In how many distinct ways can you climb to the top?

**Constraints**
- `1 <= n <= 45`

<Callout type="green" label="Core Insight">
The number of ways to reach step n equals the number of ways to reach step n-1 (then take 1 step) plus the number of ways to reach step n-2 (then take 2 steps). This recurrence is identical to the Fibonacci sequence: climbStairs(n) = climbStairs(n-1) + climbStairs(n-2).
</Callout>

<SectionHeader n={2} title="Examples" />

#### Example 1
```
Input:  n = 2
Output: 2
Explanation: Two ways — (1+1) or (2)
```

#### Example 2
```
Input:  n = 3
Output: 3
Explanation: Three ways — (1+1+1), (1+2), (2+1)
```

#### Example 3
```
Input:  n = 5
Output: 8
```

<SectionHeader n={4} title="Approach" />

Use top-down DP with a `memo` dictionary. For `n == 0` or `n == 1` return 1. Otherwise check `memo[n]`; if found return it. Otherwise compute `climbStairs(n-1) + climbStairs(n-2)`, store in `memo[n]`, and return.

**Bottom-up alternative**: Maintain just two variables `a = 1, b = 1` and iterate `n` times computing `a, b = b, a+b`. Final answer is `b`. O(n) time, O(1) space.

<Callout type="cyan" label="Fibonacci identity">
climbStairs(1)=1, climbStairs(2)=2, climbStairs(3)=3, climbStairs(4)=5, climbStairs(5)=8 — these are the Fibonacci numbers offset by 1. The problem is structurally identical to computing the nth Fibonacci number.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Base cases">
    Return 1 for `n == 0` (empty path) and `n == 1` (single step). These seed the recurrence.
  </StepItem>
  <StepItem title="Check memo">
    If `memo[n]` exists, return it immediately — avoid recomputation.
  </StepItem>
  <StepItem title="Recurse and memoize">
    Compute `result = climbStairs(n-1) + climbStairs(n-2)`, store `memo[n] = result`, return `result`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  var memo: [Int: Int] = [:]

  func climbStairs(_ n: Int) -> Int {
    if n == 0 || n == 1 {
      return 1
    }
    if let cached = memo[n] {
      return cached
    }
    let result = climbStairs(n - 1) + climbStairs(n - 2)
    memo[n] = result
    return result
  }
}
```
