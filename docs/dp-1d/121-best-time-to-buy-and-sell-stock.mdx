<SectionHeader n={1} title="Problem Overview" />

You are given an array `prices` where `prices[i]` is the price of a given stock on the `i`-th day.

You want to maximise your profit by choosing a **single day to buy** one stock and choosing a **different day in the future to sell** that stock.

Return the **maximum profit** you can achieve from this transaction. If you cannot achieve any profit, return `0`.

**Constraints:**
- `1 <= prices.length <= 10^5`
- `0 <= prices[i] <= 10^4`
- You must buy before you sell (buy on day `i`, sell on day `j > i`).

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input: prices = [7,1,5,3,6,4]
Output: 5
Explanation: Buy on day 2 (price=1), sell on day 5 (price=6). Profit = 6 - 1 = 5.
```

#### Example 2

```
Input: prices = [7,6,4,3,1]
Output: 0
Explanation: Prices only decrease; no profitable transaction is possible.
```

<SectionHeader n={4} title="Approach" />

Use two pointers starting at indices 0 and 1. At every step: if `prices[right] > prices[left]`, compute the profit and update the maximum. Otherwise, a cheaper buy day has been found — move `left` to `right`. Advance `right` each iteration.

<Callout type="cyan" label="Why move left to right on a new low">
Any future sell would yield a higher profit from the new (lower) buy price than from the old one. The old left is now strictly dominated.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Initialise pointers and result">
    `var left = 0`, `var right = 1`, `var maxProfit = 0`.
  </StepItem>
  <StepItem title="Scan with right pointer">
    While `right less than prices.count`: check if `prices[left] less than prices[right]`.
  </StepItem>
  <StepItem title="Profitable window">
    If yes, compute `profit = prices[right] - prices[left]` and update `maxProfit = max(maxProfit, profit)`.
  </StepItem>
  <StepItem title="New low buy day">
    If `prices[right] less than or equal to prices[left]`, set `left = right` — found a cheaper buy.
  </StepItem>
  <StepItem title="Advance right">
    `right += 1` each iteration. Return `maxProfit` after the loop.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func maxProfit(_ prices: [Int]) -> Int {
    var left = 0
    var right = 1
    var maxProfit = 0

    while right < prices.count {
      if prices[left] < prices[right] {
        let profit = prices[right] - prices[left]
        maxProfit = max(maxProfit, profit)
      } else {
        left = right
      }
      right += 1
    }

    return maxProfit
  }
}
```
