<SectionHeader n={1} title="Problem Overview" />

There are `n` cars at given miles away from the starting mile `0`, traveling to their destination at `target` miles.

You are given two integer arrays `position` and `speed`, both of length `n`, where `position[i]` is the position of the `i`-th car and `speed[i]` is the speed of the `i`-th car (in miles per hour).

A car can never pass another car, but it can catch up to it and travel next to it at the speed of the slower car. A **car fleet** is a non-empty set of cars driving at the same position and same speed. Note that a single car is also a car fleet.

If a car catches up to a car fleet at the moment the fleet reaches the destination, it does **not** count as a fleet.

Return the **number of car fleets** that will arrive at the destination.

**Constraints:**
- `n == position.length == speed.length`
- `1 <= n <= 10^5`
- `0 < target <= 10^6`
- `0 < position[i] < target`
- `0 < speed[i] <= 10^6`
- All values of `position` are unique.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  target = 12, position = [10,8,0,5,3], speed = [2,4,1,1,3]
Output: 3
Explanation:
  Car at 10 takes (12-10)/2 = 1.0 hr. Car at 8 takes (12-8)/4 = 1.0 hr — same, merge into fleet.
  Car at 5 takes (12-5)/1 = 7.0 hr. Car at 3 takes (12-3)/3 = 3.0 hr. Car at 0 takes 12.0 hr.
  Fleets: {10,8}, {5}, {3}, {0} — wait, car 3 is slower than car 5 ahead, so 3 does not catch 5 → 3 fleets.
```

#### Example 2

```
Input:  target = 10, position = [3], speed = [3]
Output: 1
```

<SectionHeader n={4} title="Approach" />

<CarFleetDiagram />

Sort cars by position in descending order (closest to target first). For each car, compute `time = (target - position) / speed` (as a Float). Push onto a stack. If `stack.last <= stack[stack.count-2]`, the current car catches the car ahead — pop it (they form the same fleet). Final stack size is the number of distinct fleets.

<Callout type="cyan" label="Arrival time as fleet identity">
Two cars form the same fleet if the trailing car arrives no later than the leading car. The leading car sets the pace — the trailing car catches up before or exactly at the target. Processing front-to-back, the stack always holds one representative time per distinct fleet.
</Callout>

<StepList>
  <StepItem title="Guard single car">
    `guard position.count > 1 else { return 1 }`.
  </StepItem>
  <StepItem title="Sort by position descending">
    `let cars = zip(position, speed).sorted { $0.0 > $1.0 }`.
  </StepItem>
  <StepItem title="Compute arrival times and build stack">
    For each `car`: `let time = Float(target - car.0) / Float(car.1)`. Push `time`. While `stack.count >= 2` and `stack.last! <= stack[stack.count-2]`: pop.
  </StepItem>
  <StepItem title="Return fleet count">
    `return stack.count`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func carFleet(_ target: Int, _ position: [Int], _ speed: [Int]) -> Int {
    guard position.count > 1 else { return 1 }

    let cars = Array(zip(position, speed)).sorted { $0.0 > $1.0 }
    var stack = [Float]()

    for car in cars {
      let time = (Float(target) - Float(car.0)) / Float(car.1)
      stack.append(time)
      while stack.count >= 2, stack[stack.count - 1] <= stack[stack.count - 2] {
        stack.removeLast()
      }
    }

    return stack.count
  }
}
```
