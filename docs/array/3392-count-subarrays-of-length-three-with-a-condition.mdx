<SectionHeader n={1} title="Problem Overview" />

Given an integer array `nums`, return the number of subarrays of length 3 such that the **sum of the first and third numbers equals exactly half** of the second number.

Formally, for subarray `[a, b, c]`: the condition is `a + c == b / 2`.

**Constraints:**
- `3 <= nums.length <= 100`
- `-100 <= nums[i] <= 100`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  nums = [1,2,1,4,1]
Output: 1
Explanation:
  [1,2,1]: a+c = 1+1 = 2, b/2 = 2/2 = 1. 2 ≠ 1 → no.
  [2,1,4]: a+c = 2+4 = 6, b/2 = 1/2 = 0. 6 ≠ 0 → no.
  [1,4,1]: a+c = 1+1 = 2, b/2 = 4/2 = 2. 2 == 2 → yes!
  Count = 1.
```

#### Example 2

```
Input:  nums = [1,1,1]
Output: 0
Explanation:
  [1,1,1]: a+c = 1+1 = 2, b/2 = 1/2 = 0. 2 ≠ 0 → no.
  Count = 0.
```

<SectionHeader n={4} title="Approach" />

Iterate index `i` from `0` to `nums.count - 3` inclusive. For each triple `(a, b, c) = (nums[i], nums[i+1], nums[i+2])`: check if `(a + c) * 2 == b`. Increment count on match.

<Callout type="yellow" label="Avoid float comparison">
The condition is `a + c == b / 2`. Using floating-point (`Float(b) / 2 == Float(a+c)`) works but risks precision issues. Prefer the integer form: `(a + c) * 2 == b`. This is exact for all integer inputs.
</Callout>

<StepList>
  <StepItem title="Iterate triples">
    For `i` in `0..<nums.count - 2`: extract `a = nums[i]`, `b = nums[i+1]`, `c = nums[i+2]`.
  </StepItem>
  <StepItem title="Check condition">
    If `(a + c) * 2 == b`, increment `result`.
  </StepItem>

  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func countSubarrays(_ nums: [Int]) -> Int {
    var result = 0
    for i in 0..<nums.count - 2 {
      let a = nums[i], b = nums[i + 1], c = nums[i + 2]
      if (a + c) * 2 == b {
        result += 1
      }
    }
    return result
  }
}
```
