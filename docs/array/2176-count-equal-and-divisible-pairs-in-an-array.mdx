<SectionHeader n={1} title="Problem Overview" />

Given a **0-indexed** integer array `nums` of length `n` and an integer `k`, return the **number of pairs** `(i, j)` where `0 <= i < j < n`, such that `nums[i] == nums[j]` and `(i * j)` is divisible by `k`.

**Constraints:**
- `1 <= nums.length <= 100`
- `1 <= nums[i] <= 100`
- `1 <= k <= 100`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  nums = [3,1,2,2,2,1,3], k = 2
Output: 4
Explanation: Pairs: (0,6)=3*3, (2,3)=2*2, (2,4)=2*2, (3,4)=2*2 — check (i*j)%k==0.
```

#### Example 2

```
Input:  nums = [1,2,3,4], k = 1
Output: 0
Explanation: No two elements are equal.
```

<SectionHeader n={4} title="Approach" />

Maintain a dictionary mapping each value to its list of seen indices. For each new index `i`, look up prior indices `j` with the same value and count pairs where `(i * j) % k == 0`. Then append `i` to the list.

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Initialise dict">
    `var dict = [Int: [Int]]()` mapping value to list of prior indices.
  </StepItem>
  <StepItem title="Iterate with index">
    For each `i`, look up `dict[nums[i]]`. For each stored `j`, check `(i * j) % k == 0` — if so, increment result.
  </StepItem>
  <StepItem title="Store current index">
    Append `i` to `dict[nums[i]]`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func countPairs(_ nums: [Int], _ k: Int) -> Int {
    var dict = [Int: [Int]]()
    var result = 0
    for i in 0..<nums.count {
      let num = nums[i]
      if let existing = dict[num] {
        for j in existing {
          if (i * j) % k == 0 { result += 1 }
        }
        dict[num]!.append(i)
      } else {
        dict[num] = [i]
      }
    }
    return result
  }
}
```
