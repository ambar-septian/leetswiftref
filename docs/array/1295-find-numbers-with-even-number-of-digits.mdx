<SectionHeader n={1} title="Problem Overview" />

Given an array `nums` of integers, return how many of them contain an **even number of digits**.

**Constraints:**
- `1 <= nums.length <= 500`
- `1 <= nums[i] <= 10^5`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  nums = [12, 345, 2, 6, 7896]
Output: 2
Explanation: 12 has 2 digits (even). 345 has 3. 2 has 1. 6 has 1. 7896 has 4 (even). Count = 2.
```

#### Example 2

```
Input:  nums = [555, 901, 482, 1771]
Output: 1
Explanation: Only 1771 has 4 digits (even).
```

<SectionHeader n={4} title="Approach" />

For each number, compute `Int(log10(Double(num))) + 1` to get the digit count, then check if it is even. For example: `log10(100) = 2`, so `Int(2) + 1 = 3` digits — correct.

<Callout type="green" label="Why log10 works">
A number with d digits satisfies 10^(d-1) is at most n, and n is less than 10^d. So log10(n) is in [d-1, d), and Int(log10(n)) = d-1, giving digit count = d.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Iterate through nums">
    For each index `i`, access `nums[i]`.
  </StepItem>
  <StepItem title="Compute digit count">
    `let value = Int(log10(Double(num))) + 1`
  </StepItem>
  <StepItem title="Check parity">
    If `value % 2 == 0`, increment `result`.
  </StepItem>
  <StepItem title="Return result">
    Return the count of numbers with an even digit count.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
import Foundation

class Solution {
  func findNumbers(_ nums: [Int]) -> Int {
    var result = 0
    for num in nums {
      let digits = Int(log10(Double(num))) + 1
      result += (digits % 2 == 0 ? 1 : 0)
    }
    return result
  }
}
```
