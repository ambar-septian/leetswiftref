<SectionHeader n={1} title="Problem Overview" />

There are `n` kids with candies. You are given an integer array `candies`, where each `candies[i]` represents the number of candies the `i`-th kid has, and an integer `extraCandies`, denoting the number of extra candies that you have.

Return a boolean array `result` of length `n`, where `result[i]` is `true` if, after giving the `i`-th kid all the `extraCandies`, they will have the **greatest** number of candies among all the kids, or `false` otherwise.

Note that **multiple** kids can have the greatest number of candies.

**Constraints:**
- `n == candies.length`
- `2 <= n <= 100`
- `1 <= candies[i] <= 100`
- `1 <= extraCandies <= 50`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  candies = [2,3,5,1,3], extraCandies = 3
Output: [true,true,true,false,true]
Explanation:
  Kid 0: 2+3=5 >= 5 → true
  Kid 1: 3+3=6 >= 5 → true
  Kid 2: 5+3=8 >= 5 → true
  Kid 3: 1+3=4 < 5  → false
  Kid 4: 3+3=6 >= 5 → true
```

#### Example 2

```
Input:  candies = [4,2,1,1,2], extraCandies = 1
Output: [true,false,false,false,false]
```

<SectionHeader n={4} title="Approach" />

Find the current maximum with `candies.max()!`. Then for each kid, check if `candies[i] + extraCandies >= max`. If so, they can have the greatest number of candies.

<Callout type="green" label="Key insight">
We do not change other kids' candy counts. Only one kid receives the extra at a time. So the only comparison needed is against the current overall maximum.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Find the current maximum">
    `let max = candies.max()!`
  </StepItem>
  <StepItem title="Build result array">
    For each index `i`, set `result[i] = (candies[i] + extraCandies >= max)`.
  </StepItem>
  <StepItem title="Return result">
    Return the boolean array.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func kidsWithCandies(_ candies: [Int], _ extraCandies: Int) -> [Bool] {
    let max = candies.max()!
    return candies.map { $0 + extraCandies >= max }
  }
}
```
