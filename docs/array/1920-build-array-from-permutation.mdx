<SectionHeader n={1} title="Problem Overview" />

Given a **zero-based permutation** `nums` (0-indexed), build an array `ans` of the same length where `ans[i] = nums[nums[i]]` for each `0 <= i < nums.length` and return it.

A zero-based permutation `nums` is an array of distinct integers from `0` to `nums.length - 1` (inclusive).

**Constraints:**
- `1 <= nums.length <= 1000`
- `0 <= nums[i] < nums.length`
- The elements in `nums` are distinct.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  nums = [0,2,1,5,3,4]
Output: [0,1,2,4,5,3]
Explanation: ans[0]=nums[nums[0]]=nums[0]=0, ans[1]=nums[nums[1]]=nums[2]=1, ...
```

#### Example 2

```
Input:  nums = [5,0,1,2,3,4]
Output: [4,5,0,1,2,3]
```

<SectionHeader n={4} title="Approach" />

For each index `i`, append `nums[nums[i]]` to the result. The permutation guarantee ensures `nums[i]` is always a valid index.

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Iterate and double-index">
    For each `i` from 0 to `nums.count - 1`, append `nums[nums[i]]` to the result.
  </StepItem>
  <StepItem title="Return result">
    Return the built array.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func buildArray(_ nums: [Int]) -> [Int] {
    var results = [Int]()
    for i in 0..<nums.count {
      results.append(nums[nums[i]])
    }
    return results
  }
}
```
