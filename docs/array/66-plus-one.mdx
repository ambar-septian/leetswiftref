<SectionHeader n={1} title="Problem Overview" />

You are given a **large integer** represented as an integer array `digits`, where each `digits[i]` is the `i`-th digit of the integer. The digits are ordered from most significant to least significant in left-to-right order. The large integer does not contain any leading zeros.

Increment the large integer by one and return the resulting array of digits.

**Constraints**
- `1 <= digits.length <= 100`
- `0 <= digits[i] <= 9`
- `digits` does not contain any leading zeros.

<Callout type="green" label="Core Insight">
Walk backward through the digits. If the last digit is not 9, simply increment it and stop. If it is 9, set it to 0 and carry — keep going left while digits are 9. If all digits were 9, every digit is now 0 and we need to insert a leading 1.
</Callout>

<SectionHeader n={2} title="Examples" />

#### Example 1
```
Input:  digits = [1,2,3]
Output: [1,2,4]
```

#### Example 2
```
Input:  digits = [4,3,2,1]
Output: [4,3,2,2]
```

#### Example 3
```
Input:  digits = [9]
Output: [1,0]
```

#### Example 4
```
Input:  digits = [9,9,9]
Output: [1,0,0,0]
```

<SectionHeader n={4} title="Approach" />

Iterate from the last index down to 0. If `digits[last] != 9`, increment it and immediately return. If it is 9, set it to 0 and continue left (carry propagates). After the loop exits — which means we fell off the left end, i.e., all digits were 9 — every digit is now 0, so insert a 1 at index 0.

<Callout type="cyan" label="Why insert(1, at: 0) is only reached in the all-nines case">
The loop only finishes without returning when every iteration sets a digit to 0 (all were 9). In that case digits is now all zeros and we need a new most-significant digit of 1. For any other input the early return fires before the loop ends.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Copy the array (Swift arrays are value types — mutate in place)">
    `var digits = digits`. Initialise `last = digits.count - 1`.
  </StepItem>
  <StepItem title="Walk backward">
    While `last >= 0`: if `digits[last] != 9`, increment and `break`. Else set `digits[last] = 0` and `last -= 1`.
  </StepItem>
  <StepItem title="Handle all-nines edge case">
    If `last < 0` after the loop, insert 1 at index 0: `digits.insert(1, at: 0)`.
  </StepItem>
  <StepItem title="Return">
    Return `digits`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func plusOne(_ digits: [Int]) -> [Int] {
    var digits = digits
    var last = digits.count - 1

    while last >= 0 {
      if digits[last] != 9 {
        digits[last] += 1
        break
      }
      digits[last] = 0
      last -= 1

      if last < 0 {
        digits.insert(1, at: 0)
      }
    }

    return digits
  }
}
```
