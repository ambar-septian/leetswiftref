<SectionHeader n={1} title="Problem Overview" />

Given two **0-indexed** integer arrays `nums1` and `nums2`, return a list `answer` of size 2 where:

- `answer[0]` is a list of all **distinct** integers in `nums1` which are **not** present in `nums2`.
- `answer[1]` is a list of all **distinct** integers in `nums2` which are **not** present in `nums1`.

Note that the integers in the lists may be returned in **any** order.

**Constraints:**
- `1 <= nums1.length, nums2.length <= 1000`
- `-1000 <= nums1[i], nums2[i] <= 1000`

<Callout type="amber" label="Key Constraint">
  Values can be negative (-1000 &lt;= nums[i] &lt;= 1000) and duplicate values may appear in each array. The output order is arbitrary — return distinct elements in any order.
</Callout>

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  nums1 = [1,2,3], nums2 = [2,4,6]
Output: [[1,3],[4,6]]
Explanation: 1 and 3 are in nums1 but not nums2. 4 and 6 are in nums2 but not nums1.
```

#### Example 2

```
Input:  nums1 = [1,2,3,3], nums2 = [1,1,2,2]
Output: [[3],[]]
```

<SectionHeader n={4} title="Approach" />

<Callout type="cyan" label="Core Insight">
  Convert both arrays to sets, then collect elements present in one set but absent from the other. Set membership is O(1), so two linear passes give O(n + m) total with no sorting or explicit deduplication.
</Callout>

Convert `nums1` and `nums2` to sets. Iterate `set1` and collect elements not in `set2` for `result[0]`. Iterate `set2` and collect elements not in `set1` for `result[1]`.

<Callout type="green" label="Set guarantees uniqueness">
Converting to Set automatically deduplicates both arrays. No need to separately deduplicate the output.
</Callout>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func findDifference(_ nums1: [Int], _ nums2: [Int]) -> [[Int]] {
    let set1 = Set(nums1)
    let set2 = Set(nums2)
    var result: [[Int]] = [[], []]
    for s in set1 { if !set2.contains(s) { result[0].append(s) } }
    for s in set2 { if !set1.contains(s) { result[1].append(s) } }
    return result
  }
}
```

<Callout type="green" label="Swift Notes">
  `Set(nums1)` deduplicates in O(n). `set2.contains(s)` is O(1) average. Using `var result: [[Int]] = [[], []]` and appending directly avoids a final `map` or `filter` call on the sets.
</Callout>

<SectionHeader n={8} title="Complexity Analysis" />

<Callout type="cyan" label="Time Complexity — O(n + m)">
  Building each set is O(n) and O(m). Each pass over a set is O(n) or O(m). All `contains` checks are O(1) average. Total: O(n + m).
</Callout>

<Callout type="green" label="Space Complexity — O(n + m)">
  Two sets of sizes n and m hold all distinct elements. The output lists hold at most n + m elements total. No other auxiliary structures are used.
</Callout>
