<SectionHeader n={1} title="Problem Overview" />

Determine if a `9 x 9` Sudoku board is valid. Only the filled cells need to be validated according to the following rules:

1. Each **row** must contain the digits `1-9` without repetition.
2. Each **column** must contain the digits `1-9` without repetition.
3. Each of the nine **3×3 sub-boxes** of the grid must contain the digits `1-9` without repetition.

**Note:**
- A Sudoku board (partially filled) could be valid but is not necessarily solvable.
- Only the filled cells need to be validated according to the mentioned rules.

**Constraints**
- `board.length == 9`
- `board[i].length == 9`
- `board[i][j]` is a digit `1-9` or `'.'`.

<Callout type="green" label="Core Insight">
One pass over all 81 cells. For each filled cell, check and insert into three sets — one for its row, one for its column, and one for its 3×3 box. The box index is derived by the formula `(row / 3) * 3 + (col / 3)`, mapping each cell to one of nine boxes.
</Callout>

<SectionHeader n={4} title="Approach" />

Initialise three arrays of 9 empty sets: `rows`, `cols`, `boxes`. Iterate over every cell `(i, j)`. Skip `'.'`. For each digit, check containment in `rows[i]`, `cols[j]`, and `boxes[(i/3)*3 + (j/3)]` — return `false` on any hit. Otherwise insert into all three. Return `true` after the full traversal.

<Callout type="cyan" label="Box index formula">
The 9 boxes are numbered 0–8 left-to-right, top-to-bottom. Row `i` falls in box-row `i/3` (integer division) and column `j` falls in box-column `j/3`. Combining: `boxIndex = (i/3)*3 + (j/3)`.
</Callout>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func isValidSudoku(_ board: [[Character]]) -> Bool {
    var rows  = Array(repeating: Set<Character>(), count: 9)
    var cols  = Array(repeating: Set<Character>(), count: 9)
    var boxes = Array(repeating: Set<Character>(), count: 9)

    for i in 0..<9 {
      for j in 0..<9 {
        let item = board[i][j]
        guard item != "." else { continue }

        if rows[i].contains(item)  { return false }
        rows[i].insert(item)

        if cols[j].contains(item)  { return false }
        cols[j].insert(item)

        let boxIndex = (i / 3) * 3 + (j / 3)
        if boxes[boxIndex].contains(item) { return false }
        boxes[boxIndex].insert(item)
      }
    }

    return true
  }
}
```
