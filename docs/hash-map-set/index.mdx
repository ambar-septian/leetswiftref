<SectionHeader n={1} title="What is a Hash Map / Set?" />

Both structures are backed by a **hash table**: an array of buckets indexed by `hash(key) % capacity`. A **good hash function** distributes keys uniformly so that each bucket holds ≈ 1 entry, giving O(1) average performance.

**Key properties:**

| Property | Hash Map (Dictionary) | Hash Set |
|---|---|---|
| Stores | key → value pairs | keys only |
| Lookup | O(1) average | O(1) average |
| Insert | O(1) average | O(1) average |
| Delete | O(1) average | O(1) average |
| Use case | Frequency count, complement lookup, grouping | Membership test, duplicate detection, visited tracking |
| Swift type | `Dictionary` | `Set` |

<Callout type="amber" label="Key Constraint">
  Hash map operations are O(1) **on average** but O(n) in the worst case (all keys collide into one bucket). For problems requiring guaranteed worst-case O(log n) lookups, use a balanced BST (Swift's `SortedDictionary` via swift-collections) instead.
</Callout>

<SectionHeader n={2} title="Terminology" />

| Term | Definition |
|---|---|
| **Hash function** | Maps a key to a bucket index: `index = hash(key) % capacity` |
| **Collision** | Two different keys map to the same bucket |
| **Chaining** | Resolve collisions by storing a linked list of entries in each bucket |
| **Open addressing** | Resolve collisions by probing to the next free bucket |
| **Load factor** | `n / capacity` — when this exceeds a threshold (~0.75) the table resizes |
| **Frequency map** | `[Element: Int]` dictionary that counts occurrences of each element |
| **Complement pattern** | Store seen elements; check if `target - current` is already in the map (Two Sum) |
| **Grouping** | Use a derived key (sorted chars, sorted digits) as map key to group anagrams or equivalents |

<Callout type="green" label="Rule of Thumb">
  If you are about to write a nested loop to check every pair or search for a complement — stop and ask: "can I store what I've seen so far in a hash map/set to make this O(1)?" Almost always yes.
</Callout>
