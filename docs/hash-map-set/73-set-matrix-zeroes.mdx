<SectionHeader n={1} title="Problem Overview" />

Given an `m x n` integer matrix `matrix`, if an element is `0`, set its entire row and entire column to `0`s. You must do it **in place**.

The key constraint is that the zeroing must be based on the **original** positions of zeros — not on zeros introduced by the operation itself. Setting a cell to zero must not cascade to zero out rows/columns that did not originally contain a zero.

**Constraints**
- `m == matrix.length`
- `n == matrix[0].length`
- `1 <= m, n <= 200`
- `-2^31 <= matrix[i][j] <= 2^31 - 1`

<Callout type="green" label="Core Insight">
Two-pass approach: first scan the matrix to record which rows and columns contain zeros into sets. Second pass: for every cell, if its row or column index is in either set, zero it out. The sets capture original zero positions before any modification.
</Callout>

<SectionHeader n={2} title="Examples" />

#### Example 1
```
Input:  [[1,1,1],[1,0,1],[1,1,1]]
Output: [[1,0,1],[0,0,0],[1,0,1]]
```

#### Example 2
```
Input:  [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
Output: [[0,0,0,0],[0,4,5,0],[0,3,1,0]]
```

<SectionHeader n={4} title="Approach" />

**Pass 1**: Iterate all cells. When `matrix[i][j] == 0`, insert `i` into `rowSet` and `j` into `colSet`.

**Pass 2**: Iterate all cells again. If `rowSet.contains(i)` or `colSet.contains(j)`, set `matrix[i][j] = 0`.

This guarantees the second pass only reads from the original zero positions captured in the sets.

<Callout type="cyan" label="O(1) space follow-up">
You can achieve O(1) extra space by using the first row and first column of the matrix itself as markers. Record whether the first row/column originally contained a zero, then use row[0] and col[0] as flags for all other rows/columns. Apply zeroing in reverse order of marking to avoid overwriting the flags prematurely.
</Callout>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func setZeroes(_ matrix: inout [[Int]]) {
    var row = Set<Int>()
    var column = Set<Int>()

    // Pass 1: record original zero positions
    for i in 0..<matrix.count {
      for j in 0..<matrix[i].count {
        if matrix[i][j] == 0 {
          row.insert(i)
          column.insert(j)
        }
      }
    }

    // Pass 2: zero out recorded rows and columns
    for i in 0..<matrix.count {
      for j in 0..<matrix[i].count {
        if row.contains(i) || column.contains(j) {
          matrix[i][j] = 0
        }
      }
    }
  }
}
```
