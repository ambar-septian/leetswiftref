<SectionHeader n={1} title="Problem Overview" />

You are given an `m x n` binary matrix `grid`. An **island** is a group of `1`s (land) connected **4-directionally** (horizontal or vertical). You may assume all four edges of the grid are surrounded by water.

The **area** of an island is the number of cells with value `1` in the island.

Return the **maximum area** of an island in `grid`. If there is no island, return `0`.

**Constraints:**
- `m == grid.length`
- `n == grid[i].length`
- `1 <= m, n <= 50`
- `grid[i][j]` is either `0` or `1`.

<SectionHeader n={4} title="Approach" />

Scan every cell. When a `1` is found, launch DFS. Inside DFS: immediately set `grid[r][c] = 0` (sink), then recurse in all 4 directions, summing the return counts. Return `1 + counts from neighbors`. Update `results = max(results, dfs(r, c))` after each call. Return `results` at the end.

<Callout type="cyan" label="Sinking vs visited array">
Setting grid[r][c] = 0 before recursing serves as an in-place visited marker. It is safe here because the problem does not ask us to preserve the grid. If the grid must be preserved, use a separate boolean visited matrix instead.
</Callout>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func maxAreaOfIsland(_ grid: [[Int]]) -> Int {
    guard !grid.isEmpty else { return 0 }
    var grid = grid
    let rows = grid.count
    let cols = grid[0].count
    let directions = [(0, 1), (0, -1), (1, 0), (-1, 0)]
    var results = 0

    func dfs(_ r: Int, _ c: Int) -> Int {
      if r < 0 || c < 0 || r >= rows || c >= cols || grid[r][c] == 0 {
        return 0
      }
      grid[r][c] = 0
      var count = 1
      for dir in directions {
        count += dfs(r + dir.0, c + dir.1)
      }
      return count
    }

    for r in 0..<rows {
      for c in 0..<cols {
        if grid[r][c] == 1 {
          results = max(results, dfs(r, c))
        }
      }
    }

    return results
  }
}
```
