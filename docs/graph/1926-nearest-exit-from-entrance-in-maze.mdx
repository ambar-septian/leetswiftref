<SectionHeader n={1} title="Problem Overview" />

You are given an `m x n` matrix `maze` (0-indexed) with empty cells (represented as `'.'`) and walls (represented as `'+'`). You are also given the `entrance` of the maze, where `entrance = [entranceRow, entranceCol]` denotes the row and column of the cell you are initially standing at.

In one step, you can move one cell **up, down, left, or right**. You cannot step into a wall cell, and you cannot step outside the maze. Your goal is to find the **nearest exit** from the entrance. An exit is an **empty cell at the border** of the maze. The entrance does not count as an exit.

Return the **number of steps** in the shortest path from the entrance to the nearest exit, or `-1` if no such path exists.

**Constraints:**
- `maze.length == m`
- `maze[i].length == n`
- `1 <= m, n <= 100`
- `maze[i][j]` is either `'.'` or `'+'`.
- `entrance.length == 2`
- `0 <= entrance[0] < m`
- `0 <= entrance[1] < n`
- `entrance` will always be an empty cell.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  maze = [["+","+",".","+"],[".",".",".","+"],["+","+","+","."]], entrance = [1,2]
Output: 1
```

#### Example 2

```
Input:  maze = [["+","+","+"],[".",".","."],["+","+","+"]], entrance = [1,2]

```

#### Example 3

```
Input:  maze = [[".","+"]], entrance = [0,0]
Output: -1
```

<SectionHeader n={4} title="Approach" />

Encode each cell as a single integer `r * cols + c`. BFS from the entrance. For each cell, check all 4 directions. If a direction goes out of bounds AND the current cell is not the entrance, the current cell is a border exit — return the current level count.

If a direction is in bounds and free (`'.'`), mark it `'+'` and enqueue. After each full level, increment the step counter.

<Callout type="yellow" label="Exit detection detail">
The exit check triggers when a direction goes out-of-bounds. This means the current cell (not the neighbour) is on the border. The check `entranceIndex != evidict` ensures we skip this exit trigger when processing the entrance itself.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Encode entrance and initialise BFS">
    Encode `entranceIndex = entrance[0] * cols + entrance[1]`. Push it into the queue. Set `results = 0`.
  </StepItem>
  <StepItem title="Process level by level">
    Snapshot `count = queue.count`. Dequeue exactly `count` cells. For each cell, decode `(r, c)`.
  </StepItem>
  <StepItem title="Check 4 directions">
    For each direction: if out-of-bounds and current cell is not the entrance, return `results`. If in-bounds and `maze[rD][cD] == "."`, mark `'+'` and enqueue `rD * cols + cD`.
  </StepItem>
  <StepItem title="Increment step count">
    After finishing the level, `results += 1`. Continue until queue is empty.
  </StepItem>
  <StepItem title="No exit found">
    If queue empties, return -1.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func nearestExit(_ maze: [[Character]], _ entrance: [Int]) -> Int {
    var maze = maze
    let dirs = [(0,1),(0,-1),(1,0),(-1,0)]
    let rows = maze.count, cols = maze[0].count
    let entranceIndex = entrance[0] * cols + entrance[1]
    var queue = [entranceIndex]
    var results = 0

    while !queue.isEmpty {
      let count = queue.count
      for _ in 0..<count {
        let evidict = queue.removeFirst()
        let r = evidict / cols, c = evidict % cols

        for dir in dirs {
          let rD = r + dir.0, cD = c + dir.1
          if rD < 0 || rD >= rows || cD < 0 || cD >= cols {
            if entranceIndex != evidict { return results }
          } else if maze[rD][cD] == "." {
            maze[rD][cD] = "+"
            queue.append(rD * cols + cD)
          }
        }
      }
      results += 1
    }
    return -1
  }
}
```
