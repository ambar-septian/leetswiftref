<SectionHeader n={1} title="Problem Overview" />

You are given an array of variable pairs `equations` and an array of real numbers `values`, where `equations[i] = [A_i, B_i]` and `values[i]` represent the equation `A_i / B_i = values[i]`. Each `A_i` or `B_i` is a string that represents a single variable.

You are also given some `queries`, where `queries[j] = [C_j, D_j]` represents the `j`-th query where you must find the result of `C_j / D_j = ?`.

Return the answers to all queries. If a single answer does not exist, return `-1.0`.

**Note:** The input is always valid. You may assume that evaluating the queries will not result in division by zero and that there is no contradiction.

**Note:** The variables that do not occur in the list of equations are undefined, so the answer cannot be determined for them.

**Constraints:**
- `1 <= equations.length <= 20`
- `equations[i].length == 2`
- `1 <= A_i.length, B_i.length <= 5`
- `values.length == equations.length`
- `0.0 < values[i] <= 20.0`
- `1 <= queries.length <= 20`
- `queries[i].length == 2`
- `1 <= C_j.length, D_j.length <= 5`
- `A_i, B_i, C_j, D_j` consist of lower case English letters and digits.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:
  equations = [["a","b"],["b","c"]], values = [2.0, 3.0]
  queries   = [["a","c"],["b","a"],["a","e"],["a","a"],["x","x"]]
Output: [6.0, 0.5, -1.0, 1.0, -1.0]
Explanation:
  a/b = 2.0, b/c = 3.0 → a/c = 6.0; b/a = 1/2.0 = 0.5; a/e unknown = -1.0; a/a = 1.0; x undefined = -1.0
```

#### Example 2

```
Input:
  equations = [["a","b"],["b","c"],["bc","cd"]], values = [1.5, 2.5, 5.0]
  queries   = [["a","c"],["c","b"],["bc","cd"],["cd","bc"]]
Output: [3.75, 0.4, 5.0, 0.2]
```

<SectionHeader n={4} title="Approach" />

Build a weighted directed adjacency list. For each equation `[a, b]` with value `k`:
- Add edge `a → (b, k)`
- Add edge `b → (a, 1/k)`

For each query `[src, target]`, run BFS from `src`. Maintain a queue of `(node, accumulated_weight)` pairs and a `visited` set. When `node == target`, return the weight. If BFS ends without finding `target`, return `-1`.

<EvaluateDivisionDiagram />

<Callout type="cyan" label="Neutral start weight">
BFS begins with weight 1.0 (the multiplicative identity). Each hop multiplies the accumulated weight by the edge weight. This naturally computes the product of all ratios along the path, which equals the division result.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Build weighted adjacency list">
    For each equation `[a, b]` with value `k`: `adjList[a].append((b, k))` and `adjList[b].append((a, 1/k))`.
  </StepItem>
  <StepItem title="Define BFS helper">
    `bfs(src, target) -> Double`. If either is absent from `adjList`, return `-1`.
  </StepItem>
  <StepItem title="Initialise BFS queue and visited set">
    `queue = [(src, 1.0)]`, `visited = {src}`.
  </StepItem>
  <StepItem title="Process queue">
    Dequeue `(item, weight)`. If `item == target`, return `weight`. For each unvisited neighbor `(next, w)`, push `(next, weight * w)` and mark visited.
  </StepItem>
  <StepItem title="Return -1 if not found">
    If the queue empties without finding target, return `-1`.
  </StepItem>
  <StepItem title="Answer all queries">
    For each query, call `bfs(query[0], query[1])` and collect results.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func calcEquation(_ equations: [[String]], _ values: [Double], _ queries: [[String]]) -> [Double] {
    var adjList = [String: [(String, Double)]]()

    for (i, equation) in equations.enumerated() {
      let a = equation[0], b = equation[1]
      let value = values[i]
      adjList[a, default: []].append((b, value))
      adjList[b, default: []].append((a, 1 / value))
    }

    func bfs(_ src: String, _ target: String) -> Double {
      if adjList[src] == nil || adjList[target] == nil {
        return -1
      }

      var queue = [(String, Double)]()
      var visited = Set<String>()

      queue.append((src, 1))
      visited.insert(src)

      while !queue.isEmpty {
        let (item, weight) = queue.removeFirst()
        if item == target { return weight }

        for neighbor in adjList[item, default: []] {
          if !visited.contains(neighbor.0) {
            visited.insert(neighbor.0)
            queue.append((neighbor.0, weight * neighbor.1))
          }
        }
      }

      return -1
    }

    return queries.map { bfs($0[0], $0[1]) }
  }
}
```
