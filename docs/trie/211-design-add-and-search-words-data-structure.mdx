<SectionHeader n={1} title="Problem Overview" />

Design a data structure that supports adding new words and searching for strings, where the search query may contain the wildcard character `'.'` which matches any letter.

Implement the `WordDictionary` class:
- `WordDictionary()` — Initialises the object.
- `void addWord(word)` — Adds `word` to the data structure.
- `bool search(word)` — Returns `true` if any string in the data structure matches `word`. `'.'` matches any single character.

**Constraints:**
- `1 <= word.length <= 25`
- `word` in `addWord` consists of lowercase English letters only.
- `word` in `search` consists of lowercase English letters or `'.'`.
- At most `10^4` calls will be made to `addWord` and `search`.
- At least one call will be made to `search`.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:
["WordDictionary","addWord","addWord","addWord","search","search","search","search"]
[[],["bad"],["dad"],["mad"],["pad"],["bad"],[".ad"],["b.."]]

Output: [null,null,null,null,false,true,true,true]
```

<SectionHeader n={4} title="Approach" />

<WordDictionaryDiagram />

`addWord` walks the trie character-by-character, creating nodes as needed, and sets `isEndOfWord = true` on the final node — identical to a standard trie insert.

`search` uses a recursive DFS helper that takes a node and a character index. For a regular character, it follows the single matching child. For `'.'`, it iterates over all children and recurses into each one, returning `true` if any subtree matches the remaining suffix.

<Callout type="cyan" label="Why DFS for wildcard">
An iterative approach would require maintaining multiple active trie positions simultaneously — a queue of states. Recursive DFS handles this naturally: each branch is a separate call frame that either succeeds or backtracks.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Define Node">
    `var children = [Character: Node]()`, `var isEndOfWord = false`.
  </StepItem>
  <StepItem title="addWord — standard trie insert">
    Walk characters, create missing nodes, set final node's `isEndOfWord = true`.
  </StepItem>
  <StepItem title="search — DFS with wildcard branch">
    Convert word to `[Character]`. Recursive helper takes `node` and `index`. Base case: `index == chars.count → return node.isEndOfWord`.
  </StepItem>
  <StepItem title="Handle '.' by branching">
    If `chars[index] == '.'`, iterate all `node.children.values`; return `true` if any child's subtree returns `true`.
  </StepItem>
  <StepItem title="Handle regular character">
    If the child for `chars[index]` exists, recurse with it; otherwise return `false`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Node {
  var children = [Character: Node]()
  var isEndOfWord = false
}

class WordDictionary {
  var root: Node

  init() { root = Node() }

  func addWord(_ word: String) {
    var curr = root
    for char in word {
      if let node = curr.children[char] { curr = node }
      else {
        curr.children[char] = Node()
        curr = curr.children[char]!
      }
    }
    curr.isEndOfWord = true
  }

  func search(_ word: String) -> Bool {
    let chars = Array(word)

    func dfs(_ curr: Node?, index: Int) -> Bool {
      guard let curr else { return false }
      if index == chars.count { return curr.isEndOfWord }

      let char = chars[index]
      if char == "." {
        for child in curr.children.values {
          if dfs(child, index: index + 1) { return true }
        }
        return false
      } else if let node = curr.children[char] {
        return dfs(node, index: index + 1)
      }
      return false
    }

    return dfs(root, index: 0)
  }
}
```
