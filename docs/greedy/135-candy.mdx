<SectionHeader n={1} title="Problem Overview" />

There are `n` children standing in a line. Each child is assigned a rating value given in the integer array `ratings`.

You are giving candies to these children subject to the following requirements:

- Each child must have **at least one candy**.
- Children with a **higher rating** than their immediate neighbors must get **more candies** than those neighbors.

Return the **minimum number of candies** you need to have to distribute the candies to the children.

**Constraints:**
- `n == ratings.length`
- `1 <= n <= 2 * 10^4`
- `0 <= ratings[i] <= 2 * 10^4`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input: ratings = [1,0,2]
Output: 5
Explanation: [2,1,2] — child 0 gets 2 (higher than child 1), child 1 gets 1, child 2 gets 2 (higher than child 1).
```

#### Example 2

```
Input: ratings = [1,2,2]
Output: 4
Explanation: [1,2,1] — child 1 gets more than child 0 (ascending). Child 2 gets 1; equal rating means no candy constraint.
```

<SectionHeader n={4} title="Approach" />

Initialise `result` as all `1`s (minimum one candy each).

**Pass 1 (L→R):** For each `i` from 1 to n-1, if `ratings[i] > ratings[i-1]`, set `result[i] = max(result[i], result[i-1] + 1)`.

**Pass 2 (R→L):** For each `i` from n-2 down to 0, if `ratings[i] > ratings[i+1]`, set `result[i] = max(result[i], result[i+1] + 1)`.

Sum all elements of `result`.

<Callout type="cyan" label="Why two passes work">
The L→R pass ensures every ascending run is satisfied from the left. The R→L pass then ensures descending runs are satisfied from the right. Taking the max at each position satisfies both constraints simultaneously.
</Callout>

<Callout type="green" label="Equal ratings">
Children with equal ratings have no candy constraint relative to each other — each can get just 1. The passes only trigger on strict inequality.
</Callout>

  <StepItem title="Left-to-right pass">
    For `i` in `1..<result.count`: if `ratings[i] > ratings[i-1]`, set `result[i] = max(result[i], result[i-1] + 1)`.
  </StepItem>

    For `i` in `stride(from: ratings.count - 2, through: 0, by: -1)`: if `ratings[i] > ratings[i+1]`, set `result[i] = max(result[i], result[i+1] + 1)`.
  </StepItem>

    Return `result.reduce(0, +)`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func candy(_ ratings: [Int]) -> Int {
    var result = Array(repeating: 1, count: ratings.count)

    // Left-to-right: enforce ascending constraint
    for i in 1..<result.count {
      if ratings[i] > ratings[i - 1] {
        result[i] = max(result[i], result[i - 1] + 1)
      }
    }

    // Right-to-left: enforce descending constraint
    for i in stride(from: ratings.count - 2, through: 0, by: -1) {
      if ratings[i] > ratings[i + 1] {
        result[i] = max(result[i], result[i + 1] + 1)
      }
    }

    return result.reduce(0, +)
  }
}
```
