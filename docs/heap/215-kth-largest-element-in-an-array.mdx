<SectionHeader n={1} title="Problem Overview" />

Given an integer array `nums` and an integer `k`, return the `k`th largest element in the array. Note that it is the `k`th largest element in sorted order, not the `k`th distinct element.

You must solve it without sorting.

**Constraints:**
- `1 <= k <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input: nums = [3,2,1,5,6,4], k = 2
Output: 5
```

#### Example 2

```
Input: nums = [3,2,3,1,2,4,5,5,6], k = 4
Output: 4
```

<SectionHeader n={4} title="Approach" />

Build a max-heap from all elements. Call `popMax()` k times; the value returned on the kth pop is the kth largest.

<Callout type="cyan" label="Swift Heap">
Swift's <code>Heap</code> type (from the swift-collections package) is a min-heap by default. To get max-heap behaviour, wrap values in a struct that reverses the comparator, or call <code>popMax()</code> which is provided on <code>Heap</code> for the maximum element.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Build a max-heap">
    Construct `Heap(nums)` — Swift's Heap supports `popMax()` to retrieve the largest element.
  </StepItem>
  <StepItem title="Pop max k times">
    Decrement `k` each time. Store the value returned by `popMax()` as `result`.
  </StepItem>
  <StepItem title="Return result">
    After `k` pops, `result` holds the kth largest element.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func findKthLargest(_ nums: [Int], _ k: Int) -> Int {
    var heap = Heap(nums)  // Swift Collections Heap; popMax() available
    var k = k
    var result = -1

    while k > 0 {
      guard let max = heap.popMax() else { continue }
      result = max
      k -= 1
    }

    return result
  }
}
```
