<SectionHeader n={1} title="Problem Overview" />

You are given an integer array `gifts` denoting the number of gifts in various piles. Every second, you do the following:

1. Choose the pile with the **maximum** number of gifts.
2. **Leave behind** `floor(sqrt(gifts[i]))` gifts in that pile (i.e., replace the pile with its floor square root).

Return the number of gifts remaining after `k` seconds.

**Constraints:**
- `1 <= gifts.length <= 10^3`
- `1 <= gifts[i] <= 10^9`
- `1 <= k <= 10^3`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  gifts = [25,64,9,4,100], k = 4
Output: 29
Explanation:
  Second 1: max=100 → replace with floor(sqrt(100))=10. gifts=[25,64,9,4,10].
  Second 2: max=64  → replace with floor(sqrt(64))=8.  gifts=[25,8,9,4,10].
  Second 3: max=25  → replace with floor(sqrt(25))=5.  gifts=[5,8,9,4,10].
  Second 4: max=10  → replace with floor(sqrt(10))=3.  gifts=[5,8,9,4,3].
  Sum = 5+8+9+4+3 = 29.
```

#### Example 2

```
Input:  gifts = [1,1,1,1], k = 4
Output: 4
Explanation: Each pile is already 1. sqrt(1)=1. No change after k rounds.
```

<SectionHeader n={4} title="Approach" />

Build a max-heap from `gifts`. Run `k` rounds: pop the maximum value, compute `floor(sqrt(max))`, push the result back. After `k` rounds, sum all values remaining in the heap.

<Callout type="green" label="Early exit">
If the maximum pile is 1, sqrt(1) = 1 — no change. You can stop early once the max is 1.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Build max-heap">
    Insert all gifts into a max-heap. Track the total sum.
  </StepItem>
  <StepItem title="Run k rounds">
    For each round: pop the maximum value `top`. Subtract `top` from sum. Compute `replacement = floor(sqrt(top))`. Push `replacement`. Add `replacement` to sum.
  </StepItem>
  <StepItem title="Return sum">
    Return the running sum after k rounds.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
import Foundation

class Solution {
  func pickGifts(_ gifts: [Int], _ k: Int) -> Int {
    // Use negated values in a min-heap to simulate a max-heap
    var heap = Heap(gifts.map { -$0 })
    var total = gifts.reduce(0, +)

    for _ in 0..<k {
      guard let negMax = heap.popMin() else { break }
      let maxVal = -negMax
      let replacement = Int(Double(maxVal).squareRoot())
      heap.insert(-replacement)
      total = total - maxVal + replacement
    }

    return total
  }
}
```
