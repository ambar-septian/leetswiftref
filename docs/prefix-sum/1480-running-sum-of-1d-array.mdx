<SectionHeader n={1} title="Problem Overview" />

Given an array `nums`, define a running sum of an array as `runningSum[i] = sum(nums[0] + nums[1] + ... + nums[i])`.

Return the running sum of `nums`.

**Constraints:**
- `1 <= nums.length <= 1000`
- `-10^6 <= nums[i] <= 10^6`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  nums = [1,2,3,4]
Output: [1,3,6,10]
```

#### Example 2

```
Input:  nums = [1,1,1,1,1]
Output: [1,2,3,4,5]
```

#### Example 3

```
Input:  nums = [3,1,2,10,1]
Output: [3,4,6,16,17]
```

<SectionHeader n={4} title="Approach" />

Maintain a running total `temp`. For each element in `nums`, add it to `temp` and append `temp` to the result array.

  </StepItem>
  <StepItem title="Iterate and accumulate">
    For each `num` in `nums`: `temp += num`, `results.append(temp)`.
  </StepItem>
  <StepItem title="Return results">
    Return the accumulated results array.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func runningSum(_ nums: [Int]) -> [Int] {
    var temp = 0
    var results = [Int]()
    for num in nums {
      temp += num
      results.append(temp)
    }
    return results
  }
}
```
