<SectionHeader n={1} title="Problem Overview" />

Given an integer array `nums`, return an array `answer` such that `answer[i]` is equal to the product of all the elements of `nums` except `nums[i]`.

The product of any prefix or suffix of `nums` is guaranteed to fit in a 32-bit integer. You must write an algorithm that runs in O(n) time and **without using the division operation**.

**Constraints:**
- `2 <= nums.length <= 10^5`
- `-30 <= nums[i] <= 30`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  nums = [1,2,3,4]
Output: [24,12,8,6]
```

#### Example 2

```
Input:  nums = [-1,1,0,-3,3]
Output: [0,0,9,0,0]
```

<SectionHeader n={4} title="Approach" />

**Pass 1 (left to right):** `results[i]` = product of all elements to the left of `i`. Use a running variable `left`, starting at 1.

**Pass 2 (right to left):** Multiply `results[i]` by the product of all elements to the right of `i`. Use a running variable `right`, starting at 1.

After both passes, `results[i]` = leftProduct[i] × rightProduct[i] = product of everything except `nums[i]`.

<Callout type="cyan" label="Why this avoids division">
Division would fail when any element is zero (division by zero). The prefix/suffix approach handles zeros naturally — the product for all positions containing a zero comes out to zero automatically.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Initialise result array">
    `var results = Array(repeating: 1, count: nums.count)`.
  </StepItem>
  <StepItem title="First pass — prefix products">
    `var left = 1`. For `i` from 0 to n-1: `results[i] *= left; left *= nums[i]`.
  </StepItem>
  <StepItem title="Second pass — suffix products">
    `var right = 1`. For `i` from n-1 down to 0: `results[i] *= right; right *= nums[i]`.
  </StepItem>
  <StepItem title="Return results">
    Each `results[i]` now holds the product of all elements except `nums[i]`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func productExceptSelf(_ nums: [Int]) -> [Int] {
    let count = nums.count
    var results = Array(repeating: 1, count: count)

    var left = 1
    for i in 0..<count {
      results[i] = results[i] * left
      left *= nums[i]
    }

    var right = 1
    for i in stride(from: count - 1, through: 0, by: -1) {
      results[i] = results[i] * right
      right *= nums[i]
    }

    return results
  }
}
```
