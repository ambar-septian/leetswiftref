<SectionHeader n={1} title="Problem Overview" />

Given an integer `n`, return a string array `answer` (1-indexed) where:
- `answer[i] == "FizzBuzz"` if `i` is divisible by both `3` and `5`.
- `answer[i] == "Fizz"` if `i` is divisible by `3`.
- `answer[i] == "Buzz"` if `i` is divisible by `5`.
- `answer[i] == i` (as a string) if none of the above conditions are true.

**Constraints:**
- `1 <= n <= 10^4`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  n = 3
Output: ["1", "2", "Fizz"]
```

#### Example 2

```
Input:  n = 5
Output: ["1", "2", "Fizz", "4", "Buzz"]
```

#### Example 3

```
Input:  n = 15
Output: ["1","2","Fizz","4","Buzz","Fizz","7","8","Fizz","Buzz","11","Fizz","13","14","FizzBuzz"]
```

<SectionHeader n={4} title="Approach" />

Map the range `1...n`. For each integer `i`, apply a priority-ordered conditional:
1. Both divisible by 3 and 5 → `"FizzBuzz"`
2. Divisible by 3 only → `"Fizz"`
3. Divisible by 5 only → `"Buzz"`
4. Otherwise → `"\(i)"`

<Callout type="cyan" label="Order of checks matters">
The FizzBuzz check (divisible by 15) must come before the individual Fizz and Buzz checks, otherwise numbers like 15 and 30 would match the first applicable single-divisor branch and never reach FizzBuzz.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Map range 1...n">
    `return (1...n).map { i in ... }`.
  </StepItem>
  <StepItem title="Check divisibility by both 3 and 5 first">
    `if i % 3 == 0 && i % 5 == 0 { return "FizzBuzz" }`.
  </StepItem>
  <StepItem title="Check divisibility by 3">
    `else if i % 3 == 0 { return "Fizz" }`.
  </StepItem>
  <StepItem title="Check divisibility by 5">
    `else if i % 5 == 0 { return "Buzz" }`.
  </StepItem>
  <StepItem title="Fallback to string of i">
    `else { return "\(i)" }`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func fizzBuzz(_ n: Int) -> [String] {
    return (1...n).map { i in
      if i % 3 == 0 && i % 5 == 0 {
        return "FizzBuzz"
      } else if i % 3 == 0 {
        return "Fizz"
      } else if i % 5 == 0 {
        return "Buzz"
      } else {
        return "\(i)"
      }
    }
  }
}
```
