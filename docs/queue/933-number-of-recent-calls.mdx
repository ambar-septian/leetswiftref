<SectionHeader n={1} title="Problem Overview" />

You have a `RecentCounter` class which counts the number of recent requests within a certain time frame.

Implement the `RecentCounter` class:
- `RecentCounter()` — Initializes the counter with zero recent requests.
- `int ping(int t)` — Adds a new request at time `t`, where `t` represents some time in milliseconds, and returns the number of requests that have happened in the inclusive range `[t - 3000, t]`. It is **guaranteed** that every call to `ping` uses a strictly larger value of `t` than the previous call.

**Constraints:**
- `1 <= t <= 10^9`
- Each test case will call `ping` with strictly increasing values of `t`.
- At most `10^4` calls will be made.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  ["RecentCounter","ping","ping","ping","ping"]
        [[],[1],[100],[3001],[3002]]
Output: [null,1,2,3,3]

Explanation:
  ping(1):    requests in [1-3000,1]=[-2999,1]. Only t=1 qualifies. Return 1.
  ping(100):  range [-2900,100]. t=1 and t=100 qualify. Return 2.
  ping(3001): range [1,3001]. t=1,100,3001 qualify. Return 3.
  ping(3002): range [2,3002]. t=1 no longer qualifies. Return 3.
```

<SectionHeader n={4} title="Approach" />

Store all ping timestamps in `pings`. Maintain `currentIndex` pointing to the oldest timestamp inside the current window. On each `ping(t)`: advance `currentIndex` while `t - pings[currentIndex] > 3000`. Append `t`. Return `pings.count - currentIndex` (number of elements from `currentIndex` to end inclusive).

<Callout type="cyan" label="Periodic cleanup">
The array grows indefinitely if currentIndex is never used for actual removal. This implementation batches removals: when currentIndex exceeds 1000, it removes all expired entries at once and resets the index to 0. This keeps memory bounded.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Initialise">
    `var pings = [Int]()`, `var currentIndex = 0`, `let const = 3000`.
  </StepItem>
  <StepItem title="Periodic cleanup">
    If `currentIndex > 1000`: `pings.removeFirst(currentIndex); currentIndex = 0`.
  </StepItem>
  <StepItem title="Advance index past stale entries">
    While `currentIndex < pings.count` and `t - pings[currentIndex] > 3000`: `currentIndex += 1`.
  </StepItem>
  <StepItem title="Append and return count">
    `pings.append(t)`. Return `pings.count - currentIndex`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class RecentCounter {
  let const = 3000
  var pings = [Int]()
  var currentIndex = 0

  init() {}

  func ping(_ t: Int) -> Int {
    if currentIndex > 1000 {
      pings.removeFirst(currentIndex)
      currentIndex = 0
    }

    while currentIndex < pings.count, t - pings[currentIndex] > const {
      currentIndex += 1
    }

    pings.append(t)
    return pings.count - currentIndex
  }
}
```
