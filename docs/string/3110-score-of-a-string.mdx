<SectionHeader n={1} title="Problem Overview" />

You are given a string `s`. The **score** of a string is defined as the sum of the **absolute difference** between the ASCII values of adjacent characters.

Return the **score** of `s`.

**Constraints:**
- `2 <= s.length <= 100`
- `s` consists only of lowercase English letters.

<Callout type="amber" label="Key Constraint">
  s consists only of lowercase English letters (ASCII 97–122). The max adjacent difference is 25, so the max total score is 99 × 25 = 2,475 — fits in Int. Force-unwrapping `asciiValue!` is safe given this constraint.
</Callout>

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  s = "hello"
Output: 13
Explanation:
  |'h'-'e'| = |104-101| = 3
  |'e'-'l'| = |101-108| = 7
  |'l'-'l'| = |108-108| = 0
  |'l'-'o'| = |108-111| = 3
  Total = 3+7+0+3 = 13
```

#### Example 2

```
Input:  s = "zaz"
Output: 50
Explanation: |'z'-'a'| = 25, |'a'-'z'| = 25. Total = 50.
```

<SectionHeader n={4} title="Approach" />

<Callout type="cyan" label="Core Insight">
  Convert the string to a `[Character]` array for O(1) indexed access, then iterate adjacent pairs accumulating `abs(asciiValue[i] - asciiValue[i-1])`. No sorting, no hash map — just a running total.
</Callout>

Convert `s` to a `[Character]` array for indexed access. Iterate from index 1 to the end. For each index `i`, accumulate `abs(Int(arr[i].asciiValue!) - Int(arr[i-1].asciiValue!))`.

<Callout type="green" label="ASCII values">
Lowercase English letters have ASCII values 97 ('a') through 122 ('z'). The maximum adjacent difference is 122 - 97 = 25.
</Callout>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func scoreOfString(_ s: String) -> Int {
    let arr = Array(s)
    var result = 0
    for i in 1..<arr.count {
      result += abs(Int(arr[i].asciiValue!) - Int(arr[i - 1].asciiValue!))
    }
    return result
  }
}
```

<Callout type="green" label="Swift Notes">
  `arr[i].asciiValue!` force-unwraps safely — constraints guarantee lowercase English letters (ASCII 97–122). `Int(...)` widens from `UInt8` to `Int` before subtraction to avoid unsigned underflow when computing `arr[i] - arr[i-1]` where the left side is smaller.
</Callout>

<SectionHeader n={8} title="Complexity Analysis" />

<Callout type="cyan" label="Time Complexity — O(n)">
  `Array(s)` is O(n). The single pass over n-1 adjacent pairs is O(n). Total: O(n) where n is the string length.
</Callout>

<Callout type="green" label="Space Complexity — O(n)">
  `Array(s)` allocates a Character array of length n. The `result` accumulator is a single Int — O(1) otherwise.
</Callout>
