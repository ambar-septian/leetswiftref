<SectionHeader n={1} title="Problem Overview" />

Given a string `s`, you can transform every letter individually to be lowercase or uppercase to create another string.

Return a list of all possible strings we could create. You may return the output in **any order**.

**Constraints:**
- `1 <= s.length <= 12`
- `s` consists of lowercase English letters, uppercase English letters, and digits.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  s = "a1b2"
Output: ["a1b2","a1B2","A1b2","A1B2"]
```

#### Example 2

```
Input:  s = "3z4"
Output: ["3z4","3Z4"]
```

<SectionHeader n={4} title="Approach" />

DFS with a `path` array. At each index `i`: append `chars[i]` to path, recurse to `i+1`, then pop. If `chars[i]` is a digit, that is the only branch. If it is a letter, also toggle its case (lower↔upper), append the toggled character, recurse to `i+1`, then pop. Collect results when `path.count == chars.count`.

<Callout type="cyan" label="Digit branching">
Digits produce exactly one branch — they are appended as-is and the function proceeds without a second recursive call. Letters always produce two branches (original case and toggled case).
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Convert s to character array">
    `var chars = Array(s)`.
  </StepItem>
  <StepItem title="Base case">
    If `path.count == chars.count`, append `String(path)` to results and return.
  </StepItem>
  <StepItem title="Original case branch">
    Append `chars[i]`, recurse `dfs(i+1)`, pop.
  </StepItem>
  <StepItem title="Toggled case branch (letters only)">
    `guard !chars[i].isNumber`. Toggle case (upper→lower or lower→upper). Append toggled char, recurse `dfs(i+1)`, pop.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func letterCasePermutation(_ s: String) -> [String] {
    var results = [String]()
    var path = [Character]()
    let chars = Array(s)

    func dfs(_ i: Int) {
      if path.count == chars.count {
        results.append(String(path))
        return
      }

      var char = chars[i]
      path.append(char)
      dfs(i + 1)
      path.removeLast()

      guard !char.isNumber else { return }

      if char.isUppercase {
        char = Character(char.lowercased())
      } else {
        char = Character(char.uppercased())
      }

      path.append(char)
      dfs(i + 1)
      path.removeLast()
    }

    dfs(0)
    return results
  }
}
```
