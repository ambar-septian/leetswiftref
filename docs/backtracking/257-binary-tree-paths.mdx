<SectionHeader n={1} title="Problem Overview" />

Given the `root` of a binary tree, return all root-to-leaf paths in any order.

A **leaf** is a node with no children.

**Constraints:**
- The number of nodes in the tree is in the range `[1, 100]`.
- `-100 <= Node.val <= 100`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input: root = [1,2,3,null,5]
Output: ["1->2->5","1->3"]
```

#### Example 2

```
Input: root = [1]
Output: ["1"]
```

<SectionHeader n={4} title="Approach" />

<BinaryTreePathsDiagram />

Maintain a `path` array. Push the current node's value as a string. If it is a leaf, join `path` with `"->"` and append to results. Otherwise recurse into left and right children. After all recursion from this node, pop the last element to restore the path for sibling branches.

<Callout type="cyan" label="Pop at the end vs pop after each child">
Popping once at the end of the function (after both children) is equivalent to popping after each child — but the single-pop placement at the bottom is cleaner. It restores the path for the caller regardless of which branch was taken.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Guard nil root">
    Return `[]` if the root is nil.
  </StepItem>
  <StepItem title="Push current node value">
    Append `String(root.val)` to `path`.
  </StepItem>
  <StepItem title="At leaf — record path">
    If `root.left == nil && root.right == nil`, append `path.joined(separator: "->")` to results and pop, then return.
  </StepItem>
  <StepItem title="Recurse into children">
    Call `dfs(root.left)` and `dfs(root.right)`.
  </StepItem>
  <StepItem title="Pop on return">
    `path.removeLast()` after both children have been explored.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func binaryTreePaths(_ root: TreeNode?) -> [String] {
    guard let root else { return [] }
    var results = [String]()
    var path    = [String]()

    func dfs(_ root: TreeNode?) {
      guard let root else { return }
      path.append(String(root.val))

      if root.left == nil, root.right == nil {
        results.append(path.joined(separator: "->"))
        path.removeLast()
        return
      }

      dfs(root.left)
      dfs(root.right)
      path.removeLast()
    }

    dfs(root)
    return results
  }
}
```
