<SectionHeader n={1} title="Problem Overview" />

Given the `root` of a binary tree and an integer `targetSum`, return **all root-to-leaf paths** where the sum of the node values in the path equals `targetSum`. Each path should be returned as a list of node values, from root to leaf.

A **leaf** is a node with no children.

**Constraints:**
- The number of nodes in the tree is in the range `[0, 5000]`.
- `-1000 <= Node.val <= 1000`
- `-1000 <= targetSum <= 1000`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input: root = [5,4,8,11,null,13,4,7,2,null,null,5,1], targetSum = 22
Output: [[5,4,11,2],[5,8,4,5]]
```

#### Example 2

```
Input: root = [1,2,3], targetSum = 5
Output: []
```

#### Example 3

```
Input: root = [1,2], targetSum = 0
Output: []
```

<SectionHeader n={4} title="Approach" />

<PathSumIIDiagram />

Use a mutable `paths` array as the running path. At each node, append the value, recurse left and right, then remove the value after both calls return. Record the path only when we hit a leaf whose cumulative sum equals `targetSum`.

<Callout type="cyan" label="Why removeLast works">
Because `paths` is shared across all recursive calls, removing the last element after returning from both children effectively backtracks to the state before visiting the current node ‚Äî classic backtracking.
</Callout>

<Callout type="green" label="Alternative: pure functional">
The `pathSumNoSideEffect` variant avoids mutation by passing the path down via return values ‚Äî each DFS call returns a list of paths, prepending the current node value. Cleaner but allocates more arrays.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Guard nil root">
    Return immediately when `root` is nil ‚Äî base case for leaf children.
  </StepItem>
  <StepItem title="Append current value">
    `paths.append(root.val)` and compute `newSum = sum + root.val`.
  </StepItem>
  <StepItem title="Check leaf condition">
    If `root.left == nil`, `root.right == nil`, and `newSum == targetSum`, append a copy of `paths` to `results`.
  </StepItem>
  <StepItem title="Recurse on children">
    Call `dfs(root.left, newSum)` then `dfs(root.right, newSum)` regardless of the leaf check ‚Äî we always need to explore both sides.
  </StepItem>
  <StepItem title="Backtrack">
    `paths.removeLast()` restores the shared path array to the state before this node was visited.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func pathSum(_ root: TreeNode?, _ targetSum: Int) -> [[Int]] {
    var results = [[Int]]()
    var paths = [Int]()

    func dfs(_ root: TreeNode?, _ sum: Int) {
      guard let root else { return }

      paths.append(root.val)
      let newSum = sum + root.val

      if root.left == nil, root.right == nil, newSum == targetSum {
        results.append(paths)
      }

      dfs(root.left, newSum)
      dfs(root.right, newSum)

      paths.removeLast()
    }

    dfs(root, 0)
    return results
  }
}
```
