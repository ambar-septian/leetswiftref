<SectionHeader n={1} title="What is Binary Search?" />

Binary search exploits **sorted order**: if the middle element is too small, everything to the left is also too small and can be discarded; if too large, everything to the right can be discarded. This halving means even a billion-element array requires at most 30 comparisons (log₂(10⁹) ≈ 30).

**Key properties:**

| Property | Value |
|---|---|
| Time complexity | O(log n) |
| Space complexity | O(1) iterative · O(log n) recursive call stack |
| Prerequisite | Array must be sorted (or have a monotone predicate) |
| Common variants | Exact match · Left boundary · Right boundary · Peak finding |

<Callout type="amber" label="Key Constraint">
  Binary search requires a **monotone** structure: there must be a clear dividing line where everything on the left satisfies the predicate and everything on the right does not (or vice versa). If the search space has no such structure, binary search does not apply.
</Callout>

<SectionHeader n={2} title="Terminology" />

| Term | Definition |
|---|---|
| **lo / left** | Inclusive lower bound of the current search space |
| **hi / right** | Inclusive upper bound of the current search space |
| **mid** | `lo + (hi - lo) / 2` — avoids integer overflow vs `(lo + hi) / 2` |
| **Exact match** | Return index of target if found, else -1 |
| **Left boundary** | Index of the first element satisfying the condition (first true in `F F F T T T`) |
| **Right boundary** | Index of the last element satisfying the condition (last true) |
| **Search on answer** | Binary search over a value range (not an array index) — "smallest k such that f(k) is true" |
| **Monotone predicate** | A function `f(x)` where once it becomes true it stays true (or vice versa) across the domain |

<Callout type="green" label="Rule of Thumb">
  If the brute-force is O(n) scan over a sorted or monotone space — binary search cuts it to O(log n). If you find yourself writing "find the smallest x such that condition(x) is true", that's binary search on the answer.
</Callout>
