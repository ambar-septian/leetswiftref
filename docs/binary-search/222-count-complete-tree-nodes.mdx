<SectionHeader n={1} title="Problem Overview" />

Given the `root` of a **complete** binary tree, return the number of nodes in the tree.

In a complete binary tree, every level except possibly the last is completely filled, and all nodes in the last level are as far left as possible. The number of nodes can range from `2^h` to `2^(h+1) - 1` where `h` is the height.

**Note:** The follow-up asks for an O(log² n) solution that exploits the complete-tree property. The DFS solution below runs in O(n).

**Constraints:**
- The number of nodes in the tree is in the range `[0, 5 * 10^4]`.
- `0 <= Node.val <= 5 * 10^4`
- The tree is guaranteed to be a complete binary tree.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input: root = [1,2,3,4,5,6]
Output: 6
```

#### Example 2

```
Input: root = []
Output: 0
```

#### Example 3

```
Input: root = [1]
Output: 1
```

<SectionHeader n={4} title="Approach" />

A single post-order DFS: if the node is nil return 0, otherwise return `1 + count(left) + count(right)`. Every node is visited exactly once.

<Callout type="cyan" label="O(log² n) optimisation">
For the complete-tree property: measure the left-spine height and the right-spine height at each node. If they are equal, the subtree is a perfect binary tree with <code>2^h - 1</code> nodes (return instantly). Otherwise recurse into both children. This reduces calls from O(n) to O(log² n).
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Base case">
    If `root` is nil, return `0`.
  </StepItem>
  <StepItem title="Recurse into subtrees">
    Compute `left = dfs(root.left)` and `right = dfs(root.right)`.
  </StepItem>
  <StepItem title="Return total">
    Return `1 + left + right`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func countNodes(_ root: TreeNode?) -> Int {
    func dfs(_ root: TreeNode?) -> Int {
      guard let root else { return 0 }
      return 1 + dfs(root.left) + dfs(root.right)
    }
    return dfs(root)
  }
}
```
