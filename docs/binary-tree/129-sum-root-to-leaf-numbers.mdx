<SectionHeader n={1} title="Problem Overview" />

You are given the `root` of a binary tree containing digits from `0` to `9` only.

Each root-to-leaf path in the tree represents a number. For example, the root-to-leaf path `1 → 2 → 3` represents the number `123`.

Return the **total sum** of all root-to-leaf numbers. Test cases are generated so that the answer will fit in a 32-bit integer.

A **leaf** is a node with no children.

**Constraints:**
- The number of nodes in the tree is in the range `[1, 1000]`.
- `0 <= Node.val <= 9`
- The depth of the tree will not exceed `10`.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input: root = [1,2,3]
Output: 25
Explanation: 1→2 = 12. 1→3 = 13. Sum = 25.
```

#### Example 2

```
Input: root = [4,9,0,5,1]
Output: 1026
Explanation: 4→9→5 = 495. 4→9→1 = 491. 4→0 = 40. Sum = 495 + 491 + 40 = 1026.
```

<SectionHeader n={4} title="Approach" />

<SumRootToLeafDiagram />

Define a recursive `dfs(node, prev)` that returns the sum of all numbers for paths through `node`. At each node, compute `curr = prev * 10 + node.val`. If the node is a leaf, return `curr`. Otherwise, return `dfs(left, curr) + dfs(right, curr)`.

<Callout type="cyan" label="The key recurrence">
Shifting the previous accumulated value left by one decimal place (<code>prev * 10</code>) then adding the current digit models exactly how numbers are formed: 4 → 9 → 5 gives (((0)*10+4)*10+9)*10+5 = 495.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Define dfs with accumulated value">
    `func dfs(_ root: TreeNode?, _ prev: Int) -> Int`. Initial call: `dfs(root, 0)`.
  </StepItem>
  <StepItem title="Guard nil">
    Return `0` if `root` is nil — no contribution from a missing child.
  </StepItem>
  <StepItem title="Compute current number">
    `let curr = prev * 10 + root.val`.
  </StepItem>
  <StepItem title="Leaf base case">
    If both children are nil, return `curr`.
  </StepItem>
  <StepItem title="Recurse and sum">
    Return `dfs(root.left, curr) + dfs(root.right, curr)`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func sumNumbers(_ root: TreeNode?) -> Int {
    func dfs(_ root: TreeNode?, _ prev: Int) -> Int {
      guard let root else { return 0 }

      let curr = prev * 10 + root.val

      if root.left == nil && root.right == nil {
        return curr
      }

      return dfs(root.left, curr) + dfs(root.right, curr)
    }

    return dfs(root, 0)
  }
}
```
