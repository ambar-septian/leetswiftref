<SectionHeader n={1} title="Problem Overview" />

Given the `root` of a binary tree, return the length of the **diameter** of the tree.

The **diameter** of a binary tree is the **length of the longest path** between any two nodes in the tree. This path may or may not pass through the root.

The length of a path between two nodes is represented by the number of **edges** between them.

**Constraints:**
- The number of nodes in the tree is in the range `[1, 10^4]`.
- `-100 <= Node.val <= 100`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  root = [1,2,3,4,5]
Output: 3
Explanation: The longest path is [4,2,1,3] or [5,2,1,3] — 3 edges.
```

#### Example 2

```
Input:  root = [1,2]
Output: 1
```

<SectionHeader n={4} title="Approach" />

Use a recursive helper that returns the **height** of the subtree rooted at each node (number of nodes on the longest root-to-leaf path). At each node, the candidate diameter is `left + right` (number of edges through this node). Update a closure-captured `result` with this value. Return `1 + max(left, right)` so the parent receives the correct height.

<DiameterBinaryTreeDiagram />

<Callout type="cyan" label="Height vs diameter — what the helper returns">
The helper returns height (used by the parent to compute diameter), NOT the diameter. The diameter is a side-effect captured in the outer result variable. Returning diameter from the helper would not give the parent the height it needs.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Initialise result variable">
    `var result = 0` — tracks the maximum diameter seen across all nodes.
  </StepItem>
  <StepItem title="Define recursive height function">
    `func recursive(_ root: TreeNode?) -> Int`. Base case: `guard let root else { return 0 }`.
  </StepItem>
  <StepItem title="Recurse post-order">
    `let left = recursive(root.left)`, `let right = recursive(root.right)`.
  </StepItem>
  <StepItem title="Update diameter">
    `result = max(result, left + right)`.
  </StepItem>
  <StepItem title="Return height">
    `return 1 + max(left, right)`.
  </StepItem>
  <StepItem title="Return result">
    Call `recursive(root)` and return `result`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func diameterOfBinaryTree(_ root: TreeNode?) -> Int {
    var result = 0

    func recursive(_ root: TreeNode?) -> Int {
      guard let root else { return 0 }
      let left  = recursive(root.left)
      let right = recursive(root.right)
      result = max(result, left + right)
      return 1 + max(left, right)
    }

    recursive(root)
    return result
  }
}
```

<CommonPitfalls
