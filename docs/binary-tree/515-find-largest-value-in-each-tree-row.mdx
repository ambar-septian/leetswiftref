<SectionHeader n={1} title="Problem Overview" />

Given the `root` of a binary tree, return an array of the largest value in each row of the tree (0-indexed).

**Constraints:**
- The number of nodes in the tree will be in the range `[0, 10^4]`.
- `-2^31 <= Node.val <= 2^31 - 1`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  root = [1,3,2,5,3,null,9]
Output: [1,3,9]
Explanation:
  Level 0: [1] → max = 1
  Level 1: [3, 2] → max = 3
  Level 2: [5, 3, 9] → max = 9
```

#### Example 2

```
Input:  root = [1,2,3]
Output: [1,3]
```

<SectionHeader n={4} title="Approach" />

Use a queue initialised with `root`. Each outer loop iteration processes one complete level: record `count = queue.count`, then dequeue exactly `count` nodes, tracking `maxValue`. After the inner loop, append `maxValue` to results. Enqueue left and right children as they are encountered.

<FindLargestValueDiagram />

<Callout type="cyan" label="Initialise maxValue to Int.min">
Node values can be negative — the constraint allows values down to -2^31. Initialising maxValue to 0 would produce wrong results for an all-negative level. Use Int.min as the sentinel.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Guard empty tree">
    `guard let root else { return [] }`. Initialise `var queue = [root]`.
  </StepItem>
  <StepItem title="Outer loop — one iteration per level">
    While `queue` is not empty: record `let count = queue.count`, `var maxValue = Int.min`.
  </StepItem>
  <StepItem title="Inner loop — process this level">
    Repeat `count` times: `let item = queue.removeFirst()`. Update `maxValue = max(maxValue, item.val)`. Append left and right children if they exist.
  </StepItem>
  <StepItem title="Record level result">
    After the inner loop, `results.append(maxValue)`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func largestValues(_ root: TreeNode?) -> [Int] {
    guard let root else { return [] }

    var queue = [root]
    var results = [Int]()

    while !queue.isEmpty {
      let count = queue.count
      var maxValue = Int.min

      for _ in 0..<count {
        let item = queue.removeFirst()
        maxValue = max(maxValue, item.val)

        if let left = item.left  { queue.append(left)  }
        if let right = item.right { queue.append(right) }
      }

      results.append(maxValue)
    }

    return results
  }
}
```
