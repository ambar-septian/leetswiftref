<SectionHeader n={1} title="Problem Overview" />

Given a binary tree, determine if it is **height-balanced**.

A height-balanced binary tree is one in which the left and right subtrees of **every node** differ in height by no more than `1`.

The check must apply at every node, not just the root. A tree where the root's subtrees have equal height but one subtree is itself unbalanced is **not** height-balanced.

**Constraints**
- The number of nodes in the tree is in the range `[0, 5000]`.
- `-10^4 <= Node.val <= 10^4`

<Callout type="green" label="Core Insight">
Return a (isBalanced, height) tuple from each node. Post-order DFS: compute left and right tuples first. A node is balanced if both subtrees are balanced AND their height difference is at most 1. Short-circuit: if either subtree is already unbalanced, propagate false immediately without further computation.
</Callout>

<SectionHeader n={2} title="Examples" />

#### Example 1
```
Input:  root = [3,9,20,null,null,15,7]
Output: true
```

#### Example 2
```
Input:  root = [1,2,2,3,3,null,null,4,4]
Output: false
```

#### Example 3
```
Input:  root = []
Output: true
```

<SectionHeader n={4} title="Approach" />

Define `dfs(_ root: TreeNode?) -> (Bool, Int)`. Nil returns `(true, 0)`. Recursively compute `left = dfs(root.left)` and `right = dfs(root.right)`. Compute:
- `balanced = abs(left.height - right.height) <= 1`
- `balancedResult = balanced && left.isBalanced && right.isBalanced`
- `height = 1 + max(left.height, right.height)`

Return `(balancedResult, height)`. Final answer: `dfs(root).0`.

<BalancedBinaryTreeDiagram />

<Callout type="cyan" label="Why the naive two-pass approach is O(n²)">
A naive approach calls isBalanced(root) which calls height(root.left) and height(root.right), each recursing the full subtree. Then it recurses isBalanced on both children — recomputing heights. For a skewed tree this is O(n) per node → O(n²) total. The tuple approach computes height once per node: O(n).
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Base case">
    Nil returns `(true, 0)` — an empty tree is balanced with height 0.
  </StepItem>
  <StepItem title="Post-order recursion">
    Compute `left = dfs(root.left)` and `right = dfs(root.right)` before processing the current node.
  </StepItem>
  <StepItem title="Compute balance and height">
    `balanced = abs(left.1 - right.1) <= 1`. `balancedResult = balanced && left.0 && right.0`. `height = 1 + max(left.1, right.1)`.
  </StepItem>
  <StepItem title="Return tuple">
    Return `(balancedResult, height)`. Extract `.0` at the call site.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func isBalanced(_ root: TreeNode?) -> Bool {
    func dfs(_ root: TreeNode?) -> (Bool, Int) {
      guard let root else { return (true, 0) }

      let left  = dfs(root.left)
      let right = dfs(root.right)

      let balanced      = abs(left.1 - right.1) <= 1
      let balancedResult = balanced && left.0 && right.0
      let height        = 1 + max(left.1, right.1)

      return (balancedResult, height)
    }
    return dfs(root).0
  }
}
```
