<SectionHeader n={1} title="Problem Overview" />

Given the `root` of a binary tree, return the average value of the nodes on each level in the form of an array. Answers within `10^-5` of the actual answer will be accepted.

**Constraints:**
- The number of nodes in the tree is in the range `[1, 10^4]`.
- `-2^31 <= Node.val <= 2^31 - 1`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  root = [3,9,20,null,null,15,7]
Output: [3.00000, 14.50000, 11.00000]
Explanation:
  Level 0: [3] → avg = 3.0
  Level 1: [9, 20] → avg = 29 / 2 = 14.5
  Level 2: [15, 7] → avg = 22 / 2 = 11.0
```

#### Example 2

```
Input:  root = [3,9,20,15,7]
Output: [3.00000, 14.50000, 11.00000]
```

<SectionHeader n={4} title="Approach" />

Standard level-order BFS with an inner loop. Before the inner loop, snapshot `count = queue.count`. Accumulate `sum` over all dequeued nodes. After the inner loop, compute `Double(sum) / Double(count)` and append to results. Enqueue children as usual.

<AverageOfLevelsDiagram />

<Callout type="cyan" label="Integer vs Double accumulation">
Accumulate sum as Int to avoid floating-point rounding during accumulation. Convert to Double only for the final division. This gives the most accurate result.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Guard empty tree">
    `guard let root else { return [] }`. Initialise `var queue = [root]`.
  </StepItem>
  <StepItem title="Outer loop — one iteration per level">
    While `queue` is not empty: `let count = queue.count`, `var sum = 0`.
  </StepItem>
  <StepItem title="Inner loop — dequeue count nodes">
    For each of the `count` nodes: `let item = queue.removeFirst()`, `sum += item.val`. Append non-nil left and right children to queue.
  </StepItem>
  <StepItem title="Compute and append average">
    `if count != 0 { results.append(Double(sum) / Double(count)) }`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func averageOfLevels(_ root: TreeNode?) -> [Double] {
    guard let root else { return [] }
    var queue = [root]
    var results = [Double]()

    while !queue.isEmpty {
      let count = queue.count
      var sum = 0

      for _ in 0..<count {
        let item = queue.removeFirst()
        sum += item.val

        if let left  = item.left  { queue.append(left)  }
        if let right = item.right { queue.append(right) }
      }

      if count != 0 {
        results.append(Double(sum) / Double(count))
      }
    }

    return results
  }
}
```
