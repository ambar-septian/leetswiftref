<SectionHeader n={1} title="Problem Overview" />

You are given two binary trees `root1` and `root2`.

Imagine that when you put one of them to cover the other, some nodes of the two trees are overlapped while the others are not. You need to merge the two trees into a new binary tree. The merge rule is that if two nodes overlap, then sum their values together as the new value of the merged node. Otherwise, the non-null node will be used as the node of the new tree.

Return the merged tree.

**Note:** The merging process must start from the root nodes of both trees.

**Constraints:**
- The number of nodes in both trees is in the range `[0, 2000]`.
- `-10^4 <= Node.val <= 10^4`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  root1 = [1,3,2,5], root2 = [2,1,3,null,4,null,7]
Output: [3,4,5,5,4,null,7]
Explanation: Overlapping nodes are summed; lone nodes are kept as-is.
```

#### Example 2

```
Input:  root1 = [1], root2 = [1,2]
Output: [2,2]
```

<SectionHeader n={4} title="Approach" />

Recurse with both root1 and root2. If either is nil, return the other (no merge needed). When both exist, add root2's value into root1, then recurse left and right, rewiring root1's children to the merged subtrees. Return root1 as the merged node.

<Callout type="cyan" label="In-place modification of root1">
The solution mutates root1's values and child pointers. This is efficient (no new allocations for overlapping nodes) but means root1 is no longer a valid standalone tree after the call. If the original trees must be preserved, allocate new TreeNode instances instead.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Base cases">
    If `root1 == nil`, return `root2`. If `root2 == nil`, return `root1`.
  </StepItem>
  <StepItem title="Sum overlapping values">
    `root1.val = root1.val + root2.val`.
  </StepItem>
  <StepItem title="Recurse left and right">
    `root1.left = dfs(root1.left, root2.left)`. `root1.right = dfs(root1.right, root2.right)`.
  </StepItem>
  <StepItem title="Return merged node">
    `return root1`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func mergeTrees(_ root1: TreeNode?, _ root2: TreeNode?) -> TreeNode? {
    func dfs(_ root1: TreeNode?, _ root2: TreeNode?) -> TreeNode? {
      if root1 == nil { return root2 }
      if root2 == nil { return root1 }

      guard let root1, let root2 else { return nil }

      root1.val  = root1.val + root2.val
      root1.left  = dfs(root1.left,  root2.left)
      root1.right = dfs(root1.right, root2.right)

      return root1
    }

    return dfs(root1, root2)
  }
}
```
