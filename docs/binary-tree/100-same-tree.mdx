<SectionHeader n={1} title="Problem Overview" />

Given the roots of two binary trees `p` and `q`, write a function to check if they are the same or not.

Two binary trees are considered the same if they are structurally identical, and the nodes have the same value. Both structure and values must match at every position.

**Constraints**
- The number of nodes in both trees is in the range `[0, 100]`.
- `-10^4 <= Node.val <= 10^4`

<Callout type="green" label="Core Insight">
Serialize both trees into preorder arrays including null sentinels (to capture structure). Two trees are identical if and only if their serialized arrays are equal. Alternatively, recurse directly: two trees match if both nodes are nil, or both are non-nil with the same value and matching subtrees.
</Callout>

<SectionHeader n={2} title="Examples" />

#### Example 1
```
Input:  p = [1,2,3], q = [1,2,3]
Output: true
```

#### Example 2
```
Input:  p = [1,2], q = [1,null,2]
Output: false
```

#### Example 3
```
Input:  p = [1,2,1], q = [1,1,2]
Output: false
```

<SectionHeader n={4} title="Approach" />

Define `dfs(_ tree: TreeNode?) -> [Int?]`. If `tree` is nil return `[nil]`. Otherwise return `[tree.val] + dfs(tree.left) + dfs(tree.right)`. Call on both `p` and `q` and compare the resulting arrays.

<SameTreeDiagram />

<Callout type="cyan" label="Direct recursive alternative">
The direct approach compares without serialising: if both nodes are nil return true; if one is nil and the other is not, return false; else return p.val == q.val and isSameTree(p.left, q.left) and isSameTree(p.right, q.right). Same complexity, no intermediate arrays.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Serialise with dfs">
    If node is nil return `[nil]`. Else return `[node.val] + dfs(node.left) + dfs(node.right)`.
  </StepItem>
  <StepItem title="Compare arrays">
    `return dfs(p) == dfs(q)`. Swift's `==` on `[Int?]` performs element-wise equality including nil comparisons.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func isSameTree(_ p: TreeNode?, _ q: TreeNode?) -> Bool {
    func dfs(_ tree: TreeNode?) -> [Int?] {
      guard let tree else { return [nil] }
      return [tree.val] + dfs(tree.left) + dfs(tree.right)
    }
    return dfs(p) == dfs(q)
  }
}
```
