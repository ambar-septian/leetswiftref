<SectionHeader n={1} title="Problem Overview" />

Given the roots of two binary trees `root` and `subRoot`, return `true` if there is a subtree of `root` with the same structure and node values as `subRoot` and `false` otherwise.

A subtree of a binary tree `tree` is a tree that consists of a node in `tree` and all of this node's descendants. The tree `tree` could also be considered as a subtree of itself.

**Constraints:**
- The number of nodes in the `root` tree is in the range `[1, 2000]`.
- The number of nodes in the `subRoot` tree is in the range `[1, 1000]`.
- `-10^4 <= root.val <= 10^4`
- `-10^4 <= subRoot.val <= 10^4`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  root = [3,4,5,1,2], subRoot = [4,1,2]
Output: true
Explanation: The subtree rooted at node 4 (left child of root) matches subRoot exactly.
```

#### Example 2

```
Input:  root = [3,4,5,1,2,null,null,null,null,0], subRoot = [4,1,2]
Output: false
Explanation: The subtree under node 4 has an extra node 0 that subRoot does not.
```

<SectionHeader n={4} title="Approach" />

Two mutually recursive functions:
- `isSubtree(root, subRoot)`: returns `true` if `isSameTree(root, subRoot)` or `isSubtree(root.left, subRoot)` or `isSubtree(root.right, subRoot)`.
- `isSameTree(a, b)`: both nil → `true`; values differ → `false`; recurse left and right.

<Callout type="cyan" label="Why subRoot == nil is always true">
An empty tree (nil) is trivially a subtree of any tree — the guard at the start of isSubtree returns true when subRoot is nil. This matches the problem definition.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Base cases for isSubtree">
    If `subRoot == nil`, return `true` (empty tree is always a subtree). If `root == nil`, return `false` (non-empty subRoot cannot match a nil root).
  </StepItem>
  <StepItem title="Check structural equality at current root">
    If `isSameTree(root, subRoot)` returns true, return `true` immediately.
  </StepItem>
  <StepItem title="Recurse into children">
    Return `isSubtree(root.left, subRoot) || isSubtree(root.right, subRoot)`.
  </StepItem>
  <StepItem title="isSameTree — both nil">
    If both are nil, return `true`.
  </StepItem>
  <StepItem title="isSameTree — values match">
    If `root?.val == subRoot?.val`, recurse: return `isSameTree(root?.left, subRoot?.left) && isSameTree(root?.right, subRoot?.right)`.
  </StepItem>
  <StepItem title="isSameTree — values differ">
    Return `false`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func isSubtree(_ root: TreeNode?, _ subRoot: TreeNode?) -> Bool {
    guard let subRoot else { return true }
    guard let root   else { return false }

    if isSameTree(root, subRoot) {
      return true
    }
    return isSubtree(root.left, subRoot) || isSubtree(root.right, subRoot)
  }

  func isSameTree(_ root: TreeNode?, _ subRoot: TreeNode?) -> Bool {
    if root == nil, subRoot == nil { return true }

    if root?.val == subRoot?.val {
      return isSameTree(root?.left,  subRoot?.left)
          && isSameTree(root?.right, subRoot?.right)
    }
    return false
  }
}
```
