<SectionHeader n={1} title="Problem Overview" />

Given a binary tree, find the **lowest common ancestor (LCA)** of two given nodes `p` and `q`.

The lowest common ancestor is defined as the lowest node in the tree that has both `p` and `q` as descendants (where a node is allowed to be a descendant of itself).

**Constraints:**
- The number of nodes in the tree is in the range `[2, 10^5]`.
- `-10^9 <= Node.val <= 10^9`
- All node values are unique.
- `p != q`
- Both `p` and `q` exist in the tree.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input: root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 1
Output: 3
Explanation: LCA of nodes 5 and 1 is 3.
```

#### Example 2

```
Input: root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 4
Output: 5
Explanation: LCA of nodes 5 and 4 is 5, since a node can be a descendant of itself.
```

<SectionHeader n={4} title="Approach" />

<LCADiagram />

Recurse post-order. At each node:
- If nil, return nil.
- If `root === p` or `root === q`, return `root` (found one of the targets).
- Recurse left and right. If both return non-nil, `root` is the LCA — return it. Otherwise return whichever side is non-nil (propagate upward).

<Callout type="cyan" label="Why returning root when root === p works">
The problem guarantees both p and q exist in the tree. If root === p, then q must be somewhere in p's subtree or above. Either way, p is the LCA (a node can be its own ancestor). Returning root early is correct.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Base case — nil node">
    Return nil immediately if root is nil.
  </StepItem>
  <StepItem title="Base case — target found">
    If `root === p` or `root === q`, return `root`. This covers the case where a node is its own ancestor.
  </StepItem>
  <StepItem title="Post-order recurse">
    `let left = lca(root.left, p, q)`, `let right = lca(root.right, p, q)`.
  </StepItem>
  <StepItem title="Both non-nil — LCA found">
    If `left != nil && right != nil`, return `root`.
  </StepItem>
  <StepItem title="Propagate non-nil side">
    Otherwise return `left ?? right` — pass the found node up the call stack.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func lowestCommonAncestor(_ root: TreeNode?, _ p: TreeNode?, _ q: TreeNode?) -> TreeNode? {
    guard let root else { return nil }

    if root === p || root === q { return root }

    let left  = lowestCommonAncestor(root.left,  p, q)
    let right = lowestCommonAncestor(root.right, p, q)

    if left != nil, right != nil { return root }
    return left ?? right
  }
}
```
