<SectionHeader n={1} title="Problem Overview" />

Given an array of positive integers `nums` and a positive integer `target`, return the **minimal length** of a subarray whose sum is greater than or equal to `target`. If there is no such subarray, return `0` instead.

**Constraints:**
- `1 <= target <= 10^9`
- `1 <= nums.length <= 10^5`
- `1 <= nums[i] <= 10^4`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input: target = 7, nums = [2,3,1,2,4,3]
Output: 2
Explanation: [4,3] has sum 7 with length 2.
```

#### Example 2

```
Input: target = 4, nums = [1,4,4]
Output: 1
```

#### Example 3

```
Input: target = 11, nums = [1,1,1,1,1,1,1,1]
Output: 0
```

<SectionHeader n={4} title="Approach" />

Use two pointers `i` (left) and `j` (right). Advance `j` and accumulate `sum`. Whenever `sum >= target`, record `j - i + 1` as a candidate for the minimum, subtract `nums[i]`, and advance `i`. This shrinks the window from the left to find the smallest valid subarray ending at or before `j`.

<Callout type="cyan" label="Why shrink inside a while loop">
A single removal might still leave <code>sum &gt;= target</code>. Using a <code>while</code> loop (not <code>if</code>) ensures we shrink as far as possible for the current right boundary, capturing the true minimum length at this position.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Initialise pointers and accumulators">
    `var i = 0`, `var sum = 0`, `var result = Int.max`.
  </StepItem>
  <StepItem title="Expand window with j">
    Iterate `j` from 0 to `nums.count - 1`. Add `nums[j]` to `sum`.
  </StepItem>
  <StepItem title="Shrink from left while valid">
    While `sum >= target`: update `result = min(result, j - i + 1)`, subtract `nums[i]`, increment `i`.
  </StepItem>
  <StepItem title="Return result">
    Return `result == Int.max ? 0 : result`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func minSubArrayLen(_ target: Int, _ nums: [Int]) -> Int {
    var result = Int.max
    var i = 0
    var sum = 0

    for j in 0..<nums.count {
      sum += nums[j]

      while sum >= target {
        sum -= nums[i]
        result = min(result, j - i + 1)
        i += 1
      }
    }

    return result == Int.max ? 0 : result
  }
}
```
