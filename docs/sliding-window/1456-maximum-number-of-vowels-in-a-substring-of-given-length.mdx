<SectionHeader n={1} title="Problem Overview" />

Given a string `s` and an integer `k`, return the **maximum number of vowel letters** in any substring of `s` with length `k`.

Vowel letters in English are `'a'`, `'e'`, `'i'`, `'o'`, and `'u'`.

**Constraints:**
- `1 <= s.length <= 10^5`
- `s` consists of lowercase English letters.
- `1 <= k <= s.length`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  s = "abciiidef", k = 3
Output: 3
Explanation: The substring "iii" contains 3 vowels.
```

#### Example 2

```
Input:  s = "aeiou", k = 2
Output: 2
```

#### Example 3

```
Input:  s = "leetcode", k = 3
Output: 2
```

<SectionHeader n={4} title="Approach" />

Slide a window of size `k` across the string. When a new character enters (index `i`), add 1 to `curr` if it is a vowel. When a character leaves (index `i - k`), subtract 1 if it was a vowel. Update `result = max(result, curr)` at each position.

<Callout type="green" label="No need to build initial window separately">
By guarding `if i >= k` before subtracting the leaving character, a single loop handles both the initial fill and subsequent slides.
</Callout>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func maxVowels(_ s: String, _ k: Int) -> Int {
    let vowels: Set<Character> = ["a", "e", "i", "o", "u"]
    let chars = Array(s)
    var result = 0
    var curr = 0

    for i in 0..<chars.count {
      if vowels.contains(chars[i]) { curr += 1 }
      if i >= k && vowels.contains(chars[i - k]) { curr -= 1 }
      result = max(result, curr)
    }

    return result
  }
}
```
