<SectionHeader n={1} title="Problem Overview" />

The **DNA sequence** is composed of a series of nucleotides abbreviated as `'A'`, `'C'`, `'G'`, and `'T'`.

Given a string `s` that represents a DNA sequence, return all the **10-letter-long sequences (substrings)** that occur more than once in a DNA molecule. You may return the answer in **any order**.

**Constraints:**
- `1 <= s.length <= 10^5`
- `s[i]` is either `'A'`, `'C'`, `'G'`, or `'T'`.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input: s = "AAAAACCCCCAAAAACCCCCCAAAAAGGGTTT"
Output: ["AAAAACCCCC","CCCCCAAAAA"]
```

#### Example 2

```
Input: s = "AAAAAAAAAAAAA"
Output: ["AAAAAAAAAA"]
```

<SectionHeader n={4} title="Approach" />

Use a fixed window of size 10 defined by a `left` and `right` String.Index. At each position, extract the 10-character `Substring` `s[left...right]`. If it is already in the `seen` set, add it to `result`. Otherwise add it to `seen`. Advance both indices by one. Convert the result set to `[String]` at the end.

<Callout type="cyan" label="Set for results">
Using a <code>Set</code> for results automatically deduplicates: if "AAAAAAAAAA" appears three times, it still appears once in the output.
</Callout>

<Callout type="green" label="Substring vs String hashing">
Storing <code>Substring</code> views in sets avoids copying the 10-character window on every step. The final <code>result.map(String.init)</code> materialises the strings only once.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Guard short input">
    If `s.count less than 10`, return `[]` — no 10-letter windows possible.
  </StepItem>
  <StepItem title="Initialise indices and sets">
    `var left = s.startIndex`, `var right = s.index(startIndex, offsetBy: 9)`. `var seen = Set(Substring)()`, `var result = Set(Substring)()`.
  </StepItem>
  <StepItem title="Slide the window">
    While `right less than s.endIndex`: extract `s[left...right]`, check and update `seen`/`result`, advance both indices.
  </StepItem>
  <StepItem title="Return result">
    `result.map(String.init)`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func findRepeatedDnaSequences(_ s: String) -> [String] {
    guard s.count >= 10 else { return [] }

    var seen   = Set<Substring>()
    var result = Set<Substring>()

    let startIndex = s.startIndex
    var left  = startIndex
    var right = s.index(startIndex, offsetBy: 9)

    while right < s.endIndex {
      let sub = s[left...right]

      if seen.contains(sub) {
        result.insert(sub)
      } else {
        seen.insert(sub)
      }

      left  = s.index(after: left)
      right = s.index(after: right)
    }

    return result.map(String.init)
  }
}
```
