<SectionHeader n={1} title="Problem Overview" />

You are given a string `s` and an integer `k`. You can choose any character of the string and change it to any other uppercase English character. You can perform this operation at most `k` times.

Return the length of the longest substring containing the same letter you can get after performing the above operations.

**Constraints:**
- `1 <= s.length <= 10^5`
- `s` consists of only uppercase English letters.
- `0 <= k <= s.length`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  s = "ABAB", k = 2
Output: 4
Explanation: Replace the two 'A's with 'B's (or vice versa).
```

#### Example 2

```
Input:  s = "AABABBA", k = 1
Output: 4
Explanation: Replace one 'A' in "AABA" or "ABBA" to get a window of 4.
```

<SectionHeader n={4} title="Approach" />

Maintain a frequency map `dict` and track `dominant` (the highest single-character count ever seen in the current window). At each step, update the frequency of `chars[right]` and update `dominant`. If `(right - left + 1) - dominant > k`, the window has too many characters to replace — shrink from the left. Update `result` with the current window size.

<Callout type="cyan" label="The dominant never decreases trick">
`dominant` is never decremented even after shrinking the window. This is safe because we only care about the maximum valid window ever achieved. A smaller `dominant` could only give a smaller window than what we already recorded.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Convert string to character array">
    `let chars = Array(s)`. Initialise `left = 0`, `right = 0`, `dominant = 0`, `result = 0`, `dict = [Character: Int]()`.
  </StepItem>
  <StepItem title="Expand right pointer">
    Increment frequency: `dict[chars[right], default: 0] += 1`. Update `dominant = max(dominant, dict[chars[right]])`.
  </StepItem>
  <StepItem title="Shrink if window is invalid">
    While `(right - left + 1) - dominant > k`: decrement `dict[chars[left]]` and increment `left`.
  </StepItem>
  <StepItem title="Update result">
    `result = max(right - left + 1, result)`. Increment `right`.
  </StepItem>
  <StepItem title="Return result">
    After the loop, return `result`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func characterReplacement(_ s: String, _ k: Int) -> Int {
    guard !s.isEmpty else { return 0 }

    let chars = Array(s)
    var left = 0
    var right = 0
    var dict = [Character: Int]()
    var result = 0
    var dominant = 0

    while right < chars.count {
      dict[chars[right], default: 0] += 1
      dominant = max(dominant, dict[chars[right], default: 0])

      while (right - left + 1) - dominant > k {
        dict[chars[left], default: 0] -= 1
        left += 1
      }

      result = max(right - left + 1, result)
      right += 1
    }

    return result
  }
}
```
