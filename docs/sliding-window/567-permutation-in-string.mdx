<SectionHeader n={1} title="Problem Overview" />

Given two strings `s1` and `s2`, return `true` if `s2` contains a permutation of `s1`, or `false` otherwise.

In other words, return `true` if one of `s1`'s permutations is a substring of `s2`.

**Constraints:**
- `1 <= s1.length, s2.length <= 10^4`
- `s1` and `s2` consist of lowercase English letters.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  s1 = "ab", s2 = "eidbaooo"
Output: true
Explanation: s2 contains "ba", a permutation of "ab".
```

#### Example 2

```
Input:  s1 = "ab", s2 = "eidboaoo"
Output: false
```

<SectionHeader n={4} title="Approach" />

Build `chars1Dict` for all of `s1` and `chars2Dict` for the first `s1.count` characters of `s2`. Check for a match. Then slide one character at a time: decrement the outgoing left character (remove if zero), increment the incoming right character. After each slide, compare the two maps.

<Callout type="cyan" label="Relation to Problem 438">
This problem is structurally identical to 438 (Find All Anagrams) — the only difference is that this returns a boolean on the first match rather than collecting all start indices.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Guard length">
    Return `false` if `s2.count less than s1.count`.
  </StepItem>
  <StepItem title="Build initial frequency maps">
    Populate `chars1Dict` for all of `s1`, and `chars2Dict` for the first `s1.count` characters of `s2`.
  </StepItem>
  <StepItem title="Check initial window">
    If `chars2Dict == chars1Dict`, return `true`.
  </StepItem>
  <StepItem title="Slide the window">
    For `i` from `s1.count` to `s2.count - 1`: remove `s2[i - s1.count]` from `chars2Dict` (prune zero entries), add `s2[i]`.
  </StepItem>
  <StepItem title="Check match after each slide">
    If `chars2Dict == chars1Dict`, return `true`. After the loop, return `false`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func checkInclusion(_ s1: String, _ s2: String) -> Bool {
    let chars1 = Array(s1)
    let chars2 = Array(s2)

    guard chars2.count >= chars1.count else { return false }

    var chars1Dict = [Character: Int]()
    var chars2Dict = [Character: Int]()

    for i in 0..<chars1.count {
      chars1Dict[chars1[i], default: 0] += 1
      chars2Dict[chars2[i], default: 0] += 1
    }

    if chars2Dict == chars1Dict { return true }

    for i in chars1.count..<chars2.count {
      let prev = i - chars1.count
      chars2Dict[chars2[prev], default: 0] -= 1
      if chars2Dict[chars2[prev]] == 0 {
        chars2Dict[chars2[prev]] = nil
      }
      chars2Dict[chars2[i], default: 0] += 1

      if chars2Dict == chars1Dict { return true }
    }

    return false
  }
}
```
