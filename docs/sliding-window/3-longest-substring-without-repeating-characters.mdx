<SectionHeader n={1} title="Problem Overview" />

Given a string `s`, find the length of the **longest substring** without repeating characters.

A **substring** is a contiguous non-empty sequence of characters within a string.

**Constraints**
- `0 <= s.length <= 5 * 10^4`
- `s` consists of English letters, digits, symbols, and spaces.

<Callout type="green" label="Core Insight">
Maintain a window `[left, right]` of characters with no duplicates. Expand right freely; when a duplicate appears, shrink from the left until the duplicate is removed. Track the max window size seen.
</Callout>

<SectionHeader n={2} title="Examples" />

#### Example 1
```
Input:  s = "abcabcbb"
Output: 3
Explanation: The answer is "abc", with length 3.
```

#### Example 2
```
Input:  s = "bbbbb"
Output: 1
Explanation: The answer is "b", with length 1.
```

#### Example 3
```
Input:  s = "pwwkew"
Output: 3
Explanation: The answer is "wke", with length 3.
Note: "pwke" is a subsequence, not a substring.
```

<SectionHeader n={4} title="Approach" />

Use two pointers `left` and `right` and a `Set` of characters currently in the window. Advance `right` one character at a time. If `chars[right]` is already in the set, remove `chars[left]` from the set and advance `left`. Otherwise insert `chars[right]` and update the max length. Continue until `right` reaches the end.

<Callout type="cyan" label="Why the set shrink is correct">
When we see a duplicate at `right`, the offending copy is somewhere in `[left, right-1]`. We remove from the left one by one until it is gone — then the window is valid again and we can continue expanding.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Convert string to character array">
    `Array(s)` gives O(1) index access to each character.
  </StepItem>
  <StepItem title="Initialise window pointers and set">
    `left = 0`, `right = 0`, `cache = Set`, `result = 0`.
  </StepItem>
  <StepItem title="Expand right">
    If `chars[right]` is not in `cache`, insert it, update `result = max(result, right - left + 1)`, and advance `right`.
  </StepItem>
  <StepItem title="Shrink left on duplicate">
    If `chars[right]` is already in `cache`, remove `chars[left]` from `cache` and advance `left`. Do not advance `right` yet.
  </StepItem>
  <StepItem title="Repeat until right reaches end">
    The loop terminates when `right == chars.count`. Return `result`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func lengthOfLongestSubstring(_ s: String) -> Int {
    let chars = Array(s)
    var cache = Set<Character>()
    var result = 0
    var left = 0
    var right = 0

    while right < chars.count {
      if !cache.contains(chars[right]) {
        cache.insert(chars[right])
        result = max(result, right - left + 1)
        right += 1
        
      } else {
        cache.remove(chars[left])
        left += 1
      }
    }

    return result
  }
}
```
