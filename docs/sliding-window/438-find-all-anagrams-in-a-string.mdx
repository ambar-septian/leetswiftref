<SectionHeader n={1} title="Problem Overview" />

Given two strings `s` and `p`, return an array of all the start indices of `p`'s anagrams in `s`. You may return the answer in any order.

An anagram is a string that contains the same characters, only the order of characters can be different.

**Constraints:**
- `1 <= s.length, p.length <= 3 * 10^4`
- `s` and `p` consist of lowercase English letters.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  s = "cbaebabacd", p = "abc"
Output: [0, 6]
Explanation:
  Substring s[0..2] = "cba" is an anagram of "abc".
  Substring s[6..8] = "bac" is an anagram of "abc".
```

#### Example 2

```
Input:  s = "abab", p = "ab"
Output: [0, 1, 2]
Explanation: "ab", "ba", "ab" are all anagrams.
```

<SectionHeader n={4} title="Approach" />

Build two 26-element frequency arrays: `countP` for all characters in `p`, and `countS` for the first `p.count` characters of `s`. If they match, record index `0`. Then slide the window one character at a time: add `s[i]` to `countS` and remove `s[i - p.count]`. After each slide, compare `countS == countP`; if equal, record the start index `i - p.count + 1`.

<Callout type="cyan" label="26-slot array vs hash map">
Using an array of size 26 (indexed by `char.asciiValue - 'a'.asciiValue`) is faster than a hash map and allows direct array equality comparison with `==`. For an alphabet-bounded problem, this is the preferred approach.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Guard length">
    Return `[]` if `s.count less than p.count`.
  </StepItem>
  <StepItem title="Build initial frequency arrays">
    Initialise `countP` and `countS` as `[Int]` of size 26. Populate both for the first `p.count` characters.
  </StepItem>
  <StepItem title="Check initial window">
    If `countP == countS`, append `0` to `result`.
  </StepItem>
  <StepItem title="Slide the window">
    For `i` from `p.count` to `s.count - 1`: add `sArray[i]`, remove `sArray[i - p.count]`.
  </StepItem>
  <StepItem title="Record match">
    After each slide, if `countS == countP`, append `i - p.count + 1`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func findAnagrams(_ s: String, _ p: String) -> [Int] {
    let sArray = Array(s)
    let pArray = Array(p)
    let sCount = sArray.count
    let pCount = pArray.count

    guard sCount >= pCount else { return [] }

    var result = [Int]()
    var countP = Array(repeating: 0, count: 26)
    var countS = countP

    func idx(_ c: Character) -> Int {
      Int(c.asciiValue! - Character("a").asciiValue!)
    }

    for i in 0..<pCount {
      countP[idx(pArray[i])] += 1
      countS[idx(sArray[i])] += 1
    }

    if countP == countS {
      result.append(0)
    }

    for i in pCount..<sCount {
      countS[idx(sArray[i])] += 1
      countS[idx(sArray[i - pCount])] -= 1

      if countS == countP {
        result.append(i - pCount + 1)
      }
    }

    return result
  }
}
```
