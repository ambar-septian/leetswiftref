<SectionHeader n={1} title="Problem Overview" />

You are given a **perfect binary tree** where all leaves are on the same level and every parent has two children. The tree node has an extra pointer `next` that should be populated to point to its **next right node**. If there is no next right node, `next` should be set to `null`.

Initially all `next` pointers are set to `null`. Populate every `next` pointer to point to its next right node on the same level.

**Constraints:**
- The number of nodes in the tree is in the range `[0, 2^12 - 1]` (a perfect binary tree).
- `-1000 <= Node.val <= 1000`
- **Follow-up:** You may only use **constant extra space** (the recursion stack does not count for this purpose).

<SectionHeader n={4} title="Approach" />

Maintain a `leftMost` pointer that starts at the root and descends one level per outer loop iteration. An inner `curr` pointer walks the current level using `curr.next`:

- `curr.left.next = curr.right` — siblings within the same parent.
- `curr.right.next = curr.next.left` — cross-parent link using the already-populated `curr.next`.

<Callout type="cyan" label="Why this is O(1) space">
We never allocate a queue. The next pointers we set on the current level become the traversal links for the next iteration of the outer loop.
</Callout>

<Callout type="green" label="BFS fallback">
A BFS solution with a queue is also valid and simpler to reason about, but uses O(n) space. The O(1) solution is preferred for this problem.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Guard empty tree">
    Return nil immediately if root is nil.
  </StepItem>
  <StepItem title="Outer loop: descend levels">
    `var leftMost = root`; loop while `leftMost.left != nil` (not yet at the leaf level).
  </StepItem>
  <StepItem title="Inner loop: connect children">
    `var curr = leftMost`; loop while `curr != nil`: set `curr.left.next = curr.right`, and if `curr.next` exists, set `curr.right.next = curr.next.left`. Advance `curr = curr.next`.
  </StepItem>
  <StepItem title="Descend to next level">
    After the inner loop finishes, `leftMost = leftMost.left` to move down one level.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func connect(_ root: Node?) -> Node? {
    guard let root else { return nil }

    var leftMost: Node? = root

    while leftMost?.left != nil {
      var curr = leftMost

      while curr != nil {
        curr?.left?.next = curr?.right

        if let next = curr?.next {
          curr?.right?.next = next.left
        }

        curr = curr?.next
      }

      leftMost = leftMost?.left
    }

    return root
  }
}
```
