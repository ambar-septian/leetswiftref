<SectionHeader n={1} title="Problem Overview" />

Given the `head` of a singly linked list, return `true` if it is a palindrome or `false` otherwise.

**Constraints:**
- The number of nodes in the list is in the range `[1, 10^5]`.
- `0 <= Node.val <= 9`

<SectionHeader n={2} title="Examples" />

<PalindromeLinkedListDiagram />

#### Example 1

```
Input: head = [1,2,2,1]
Output: true
```

#### Example 2

```
Input: head = [1,2]
Output: false
```

<SectionHeader n={4} title="Approach" />

Three phases:
1. **Find the midpoint** — move `slow` one step and `fast` two steps until `fast.next == nil` or `fast.next.next == nil`. `slow` ends at the last node of the first half.
2. **Reverse the second half** — iterative three-pointer reversal starting from `slow`, producing a reversed chain (`prev`).
3. **Compare both halves** — walk `p1` from `head` and `p2` from `prev` simultaneously; if any values differ, return `false`.

<Callout type="cyan" label="Odd vs even length">
For odd-length lists the middle node is skipped — only the nodes after it form the second half. The fast/slow condition <code>fast?.next != nil and fast?.next?.next != nil</code> correctly positions slow at the end of the first half in both cases.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Find the midpoint with fast/slow pointers">
    Move `slow` one step and `fast` two steps each iteration until `fast?.next == nil` or `fast?.next?.next == nil`.
  </StepItem>
  <StepItem title="Reverse the second half">
    Starting from `slow` (inclusive), apply the iterative reversal: save `next`, redirect `curr.next = prev`, advance both.
  </StepItem>
  <StepItem title="Compare first and second halves">
    Walk `p1` from `head` and `p2` from `prev`. Return `false` on any value mismatch.
  </StepItem>
  <StepItem title="Return true">
    If `p2` is exhausted without a mismatch, the list is a palindrome.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func isPalindrome(_ head: ListNode?) -> Bool {
    var slow = head
    var fast = head

    while fast?.next != nil && fast?.next?.next != nil {
      slow = slow?.next
      fast = fast?.next?.next
    }

    // Reverse second half starting from slow
    var prev: ListNode? = nil
    var curr = slow

    while curr != nil {
      let next = curr?.next
      curr?.next = prev
      prev = curr
      curr = next
    }

    // Compare both halves
    var p1 = head
    var p2 = prev

    while p1 != nil, p2 != nil {
      if p1?.val != p2?.val { return false }
      p1 = p1?.next
      p2 = p2?.next
    }

    return true
  }
}
```
