<SectionHeader n={1} title="Problem Overview" />

Given the `head` of a linked list, remove the `n`th node from the **end** of the list and return its head.

**Constraints**
- The number of nodes in the list is `sz`.
- `1 <= sz <= 30`
- `0 <= Node.val <= 100`
- `1 <= n <= sz`

<Callout type="green" label="Core Insight">
Use a fast pointer that moves 2 steps at a time and a slow pointer that moves 1 step. When fast stops, count gives the total node count. Convert that to a 0-based front index, walk a dummy-headed cursor to the node just before the target, then skip the target with curr.next = curr.next.next.
</Callout>

<SectionHeader n={2} title="Examples" />

#### Example 1
```
Input:  head = [1,2,3,4,5], n = 2
Output: [1,2,3,5]
Explanation: The 2nd node from the end is node 4 — remove it.
```

#### Example 2
```
Input:  head = [1], n = 1
Output: []
```

#### Example 3
```
Input:  head = [1,2], n = 1
Output: [1]
```

<SectionHeader n={4} title="Approach" />

Start `slow` at `head` and `fast` at `head.next`, `count = 1`. Advance both while `fast != nil && fast.next != nil` (fast moves 2 steps, slow moves 1 step, count increments). After the loop, `count` encodes half the list length. Compute the true length: `count * 2`, then subtract 1 if `fast` is nil (odd-length list). The front index of the target is `length - n`. Walk a dummy-headed cursor that many steps, then unlink via `curr.next = curr.next.next`.

<Callout type="amber" label="Why the count formula works">
Fast moves 2 steps per iteration. If the list has L nodes: odd L → fast ends at nil after (L-1)/2 iterations, so count = (L+1)/2 and length = 2*count - 1. Even L → fast ends at the last node after L/2-1 iterations... actually fast ends when fast.next is nil, so for even L fast stops at the last node (not nil), count = L/2, length = 2*count. The fast==nil check distinguishes the two cases.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Initialise fast/slow and count">
    `slow = head`, `fast = head?.next`, `count = 1`.
  </StepItem>
  <StepItem title="Advance to count nodes">
    While `fast != nil && fast?.next != nil`: move `slow` one step, `fast` two steps, `count += 1`.
  </StepItem>
  <StepItem title="Compute true length">
    `count *= 2`. If `fast == nil` (odd length), `count -= 1`. `target = max(0, count - n)`.
  </StepItem>
  <StepItem title="Attach dummy head">
    Create `dummy = ListNode(0)`, `curr = dummy`, `curr.next = head`.
  </StepItem>
  <StepItem title="Walk to the node before target">
    Advance `curr` exactly `target` times via `curr = curr.next`.
  </StepItem>
  <StepItem title="Unlink the target">
    `curr.next = curr.next?.next`. Return `dummy.next`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func removeNthFromEnd(_ head: ListNode?, _ n: Int) -> ListNode? {
    var slow = head
    var fast = head?.next
    var count = 1

    while fast != nil, fast?.next != nil {
      slow = slow?.next
      fast = fast?.next?.next
      count += 1
    }

    count = count * 2
    if fast == nil { count -= 1 }
    let target = max(0, count - n)

    var dummy: ListNode? = ListNode(0)
    var curr = dummy
    curr?.next = head

    for _ in 0..<target {
      curr = curr?.next
    }
    curr?.next = curr?.next?.next
    return dummy?.next
  }
}
```
