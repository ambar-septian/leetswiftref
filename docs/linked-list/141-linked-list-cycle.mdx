<SectionHeader n={1} title="Problem Overview" />

Given `head`, the head of a linked list, determine if the linked list has a **cycle** in it.

There is a cycle in a linked list if there is some node in the list that can be reached again by continuously following the `next` pointer. Internally, `pos` is used to denote the index of the node that the tail's `next` pointer is connected to (0-indexed). Note that `pos` is not passed as a parameter.

Return `true` if there is a cycle in the linked list. Otherwise, return `false`.

**Constraints:**
- The number of nodes in the list is in the range `[0, 10^4]`.
- `-10^5 <= Node.val <= 10^5`
- `pos` is `-1` (no cycle) or a valid index in the linked list.
- **Follow-up:** Can you solve it using O(1) memory?

<SectionHeader n={4} title="Approach" />

Initialise `slow = head` and `fast = head.next`. Each iteration: advance `slow` one step and `fast` two steps. If they ever point to the same node (`===`), a cycle exists. If `fast` or `fast.next` becomes nil, the list is acyclic.

<Callout type="cyan" label="Why start fast at head.next">
Starting both at <code>head</code> would immediately satisfy <code>slow === fast</code> before any movement. Starting fast one step ahead avoids this false positive.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Initialise pointers">
    `var slow = head`, `var fast = head?.next`.
  </StepItem>
  <StepItem title="Loop while both non-nil">
    `while slow != nil, fast != nil`.
  </StepItem>
  <StepItem title="Advance pointers">
    `slow = slow?.next`, `fast = fast?.next?.next`.
  </StepItem>
  <StepItem title="Check for meeting">
    After advancing, if `slow === fast`, return `true`.
  </StepItem>
  <StepItem title="No cycle">
    Loop exits when `fast` or `fast.next` is nil — return `false`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func hasCycle(_ head: ListNode?) -> Bool {
    var slow = head
    var fast = head?.next

    while slow != nil, fast != nil {
      slow = slow?.next
      fast = fast?.next?.next

      if slow === fast {
        return true
      }
    }

    return false
  }
}
```
