<SectionHeader n={1} title="Problem Overview" />

Given the `head` of a linked list and an integer `val`, remove all the nodes of the linked list that have `Node.val == val`, and return the new head.

**Constraints:**
- The number of nodes in the list is in the range `[0, 10^4]`.
- `1 <= Node.val <= 50`
- `0 <= val <= 50`

<SectionHeader n={2} title="Examples" />

<RemoveElementsDiagram />

#### Example 1

```
Input: head = [1,2,6,3,4,5,6], val = 6
Output: [1,2,3,4,5]
```

#### Example 2

```
Input: head = [], val = 1
Output: []
```

#### Example 3

```
Input: head = [7,7,7,7], val = 7
Output: []
```

<SectionHeader n={4} title="Approach" />

Prepend a dummy sentinel node before the head. Traverse with `curr` starting at the dummy. At each step, peek at `curr.next.val`: if it equals `val`, skip it by setting `curr.next = curr.next.next` (do not advance `curr`); otherwise advance `curr = curr.next`. Return `dummy.next` as the new head.

<Callout type="cyan" label="Why a dummy head">
Without a dummy, removing the head node requires a special case. The dummy absorbs it: curr always has a predecessor, so every node (including the original head) is handled uniformly.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Create a dummy sentinel">
    Allocate a new `ListNode()` as `curr` and save `result = curr`. Point `curr.next = head`.
  </StepItem>
  <StepItem title="Traverse until curr has no next">
    Loop while `curr != nil`. The loop ends when `curr` itself becomes nil (after advancing past the last real node).
  </StepItem>
  <StepItem title="Check curr.next.val">
    If `curr.next?.val == val`, set `curr.next = curr.next?.next` (bypass the matching node). Do not advance `curr`.
  </StepItem>
  <StepItem title="Advance if no match">
    Otherwise, move `curr = curr.next`.
  </StepItem>
  <StepItem title="Return dummy.next">
    Return `result?.next` — the first real node after the sentinel.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func removeElements(_ head: ListNode?, _ val: Int) -> ListNode? {
    var curr: ListNode? = ListNode()
    var result = curr
    curr?.next = head

    while curr != nil {
      if curr?.next?.val == val {
        curr?.next = curr?.next?.next
      } else {
        curr = curr?.next
      }
    }

    return result?.next
  }
}
```
