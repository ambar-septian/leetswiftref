<SectionHeader n={1} title="Problem Overview" />

A linked list of length `n` is given such that each node contains an additional random pointer, which could point to any node in the list, or `null`.

Construct a **deep copy** of the list. The deep copy should consist of exactly `n` brand new nodes, where each new node has its value set to the value of its corresponding original node. Both the `next` and `random` pointers of the new nodes should point to new nodes in the copied list — no pointers in the new list should point to nodes in the original list.

Return the head of the copied linked list.

**Constraints:**
- `0 <= n <= 1000`
- `-10^4 <= Node.val <= 10^4`
- `Node.random` is `null` or points to some node in the linked list.

<SectionHeader n={4} title="Approach" />

**Pass 1 — Interleave:** For each original node, create a clone and insert it immediately after the original: `clone.next = orig.next; orig.next = clone`.

**Pass 2 — Random pointers:** For each original node, `orig.next.random = orig.random?.next`. The clone of any node is always at `node.next`.

**Pass 3 — Detach:** Walk the interleaved list, restoring original `next` pointers and extracting the clone list simultaneously.

<Callout type="cyan" label="Why interleaving works">
After pass 1, every original node's clone is one step ahead. So to set a clone's random, the random of the original node also has its clone one step ahead — no hash map needed.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Pass 1: insert clones">
    Walk original list. For each node: `clone.next = curr.next; curr.next = clone; curr = clone.next`.
  </StepItem>
  <StepItem title="Pass 2: wire random pointers">
    Walk again. For each original node: `node.next?.random = node.random?.next`. The clone of `node.random` is always at `node.random.next`.
  </StepItem>
  <StepItem title="Pass 3: detach">
    Use a dummy head for the clone list. For each original node: extract `clone = orig.next`, `nextOrig = clone.next`. Set `orig.next = nextOrig`, append `clone` to clone list.
  </StepItem>
  <StepItem title="Return clone head">
    Return `headCopy?.next` (skip the dummy sentinel).
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func copyRandomList(_ head: Node?) -> Node? {
    // Pass 1: interleave clones
    var curr = head
    while curr != nil {
      let clone = Node(curr!.val)
      clone.next = curr?.next
      curr?.next = clone
      curr = clone.next
    }

    // Pass 2: set random pointers
    curr = head
    while let node = curr {
      node.next?.random = node.random?.next
      curr = node.next?.next
    }

    // Pass 3: detach
    let headCopy: Node? = Node(0)
    var copy = headCopy
    curr = head
    while let orig = curr {
      let clone    = orig.next
      let nextOrig = clone?.next

      copy?.next = clone
      copy = clone

      orig.next = nextOrig
      curr = nextOrig
    }

    return headCopy?.next
  }
}
```
