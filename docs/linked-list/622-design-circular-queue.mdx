<SectionHeader n={1} title="Problem Overview" />

Design your implementation of the circular queue. The circular queue is a linear data structure in which the operations are performed based on the FIFO (First In First Out) principle, and the last position is connected back to the first position to make a circle (also called a "Ring Buffer").

Implement the `MyCircularQueue` class:

- `MyCircularQueue(k)` â€” Initializes the object with the size of the queue to be `k`.
- `boolean enQueue(int value)` â€” Inserts an element into the circular queue. Returns `true` if the operation is successful.
- `boolean deQueue()` â€” Deletes an element from the circular queue. Returns `true` if the operation is successful.
- `int Front()` â€” Gets the front item from the queue. If the queue is empty, returns `-1`.
- `int Rear()` â€” Gets the last item from the queue. If the queue is empty, returns `-1`.
- `boolean isEmpty()` â€” Checks whether the circular queue is empty or not.
- `boolean isFull()` â€” Checks whether the circular queue is full or not.

You must solve the problem without using the built-in queue data structure in your programming language.

**Constraints:**
- `1 <= k <= 1000`
- `0 <= value <= 1000`
- At most `3000` calls will be made to `enQueue`, `deQueue`, `Front`, `Rear`, `isEmpty`, and `isFull`.

<SectionHeader n={2} title="Examples" />

#### Example 1

```
MyCircularQueue cq = new MyCircularQueue(3);
cq.enQueue(1);  // true
cq.enQueue(2);  // true
cq.enQueue(3);  // true
cq.enQueue(4);  // false (full)
cq.Rear();      // 3
cq.isFull();    // true
cq.deQueue();   // true
cq.enQueue(4);  // true
cq.Rear();      // 4
```

<SectionHeader n={4} title="Approach" />

Allocate `items: [Int?]` of size `maxCount`. Maintain `head` (index of front element), `tail` (index of next write slot), and `count`. For `enQueue`: place at `items[tail]`, advance `tail = (tail + 1) % maxCount`, increment count. For `deQueue`: clear `items[head]`, advance `head = (head + 1) % maxCount`, decrement count. `Rear()` reads from `(tail - 1 + maxCount) % maxCount`.

<CircularQueueDiagram />

<Callout type="cyan" label="Why tail needs +maxCount before modulo">
(tail - 1) can be -1 when tail == 0. In Swift (and most languages), -1 % 5 = -1, not 4. Adding maxCount before the modulo guarantees a non-negative result: (0 - 1 + 5) % 5 = 4.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Initialise fields">
    `head = 0`, `tail = 0`, `count = 0`, `items = [Int?](repeating: nil, count: maxCount)`.
  </StepItem>
  <StepItem title="enQueue â€” guard full, write, advance tail">
    If `count == maxCount`, return `false`. Else: `items[tail] = value`, `tail = (tail + 1) % maxCount`, `count += 1`, return `true`.
  </StepItem>
  <StepItem title="deQueue â€” guard empty, clear, advance head">
    If `count == 0`, return `false`. Else: `items[head] = nil`, `head = (head + 1) % maxCount`, `count -= 1`, return `true`.
  </StepItem>
  <StepItem title="Front â€” guard empty, read head">
    `guard let value = items[head] else { return -1 }; return value`.
  </StepItem>
  <StepItem title="Rear â€” guard empty, read tail-1 with wrap">
    `let realTail = (tail - 1 + maxCount) % maxCount; guard let value = items[realTail] else { return -1 }; return value`.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class MyCircularQueue {
  var head = 0
  var tail = 0
  var count = 0
  var items: [Int?] = []
  let maxCount: Int

  init(_ k: Int) {
    self.maxCount = k
    self.items = Array(repeating: nil, count: maxCount)
  }

  func enQueue(_ value: Int) -> Bool {
    guard count < maxCount else { return false }
    items[tail] = value
    tail = (tail + 1) % maxCount
    count += 1
    return true
  }

  func deQueue() -> Bool {
    guard count > 0 else { return false }
    items[head] = nil
    head = (head + 1) % maxCount
    count -= 1
    return true
  }

  func Front() -> Int {
    guard let value = items[head] else { return -1 }
    return value
  }

  func Rear() -> Int {
    let realTail = (tail - 1 + maxCount) % maxCount
    guard let value = items[realTail] else { return -1 }
    return value
  }

  func isEmpty() -> Bool { count == 0 }
  func isFull() -> Bool  { count == maxCount }
}
```
