<SectionHeader n={1} title="Problem Overview" />

Given an integer `num`, return the number of steps to reduce it to zero.

In one step, if the current number is even, you have to divide it by 2; otherwise, you have to subtract 1 from it.

**Constraints:**
- `0 <= num <= 10^6`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input:  num = 14
Output: 6
Explanation:
  14 is even → 14/2 = 7        (step 1)
  7  is odd  → 7-1  = 6        (step 2)
  6  is even → 6/2  = 3        (step 3)
  3  is odd  → 3-1  = 2        (step 4)
  2  is even → 2/2  = 1        (step 5)
  1  is odd  → 1-1  = 0        (step 6)
```

#### Example 2

```
Input:  num = 8
Output: 4
Explanation: 8→4→2→1→0 (4 steps, all divisions).
```

#### Example 3

```
Input:  num = 123
Output: 12
```

<SectionHeader n={4} title="Approach" />

Simulate the process directly: while `current != 0`, apply the appropriate operation and count steps. This is O(log n) because each division by 2 halves the number.

<Callout type="cyan" label="Bit insight">
In binary, dividing by 2 is a right shift (removes the trailing 0). Subtracting 1 from an odd number clears its trailing 1. So each bit in the binary representation of num contributes exactly the steps needed to remove it.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Initialise">
    Set `current = num`, `result = 0`.
  </StepItem>
  <StepItem title="Loop until zero">
    While `current != 0`: if `current % 2 == 0` divide by 2; otherwise subtract 1. Increment `result` each iteration.
  </StepItem>
  <StepItem title="Return result">
    Return the total step count.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func numberOfSteps(_ num: Int) -> Int {
    var result = 0
    var current = num
    while current != 0 {
      current = current % 2 == 0 ? current / 2 : current - 1
      result += 1
    }
    return result
  }
}
```
