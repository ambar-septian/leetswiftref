<SectionHeader n={1} title="Problem Overview" />

Given the `root` of a binary search tree and an integer `k`, return the `k`th smallest value (1-indexed) among all the values of the nodes in the tree.

A BST's inorder traversal yields values in ascending sorted order. The kth node visited during inorder traversal is the kth smallest.

**Constraints:**
- The number of nodes in the tree is `n`.
- `1 <= k <= n <= 10^4`
- `0 <= Node.val <= 10^4`

<SectionHeader n={2} title="Examples" />

#### Example 1

```
Input: root = [3,1,4,null,2], k = 1
Output: 1
```

#### Example 2

```
Input: root = [5,3,6,2,4,null,null,1], k = 3
Output: 3
```

<SectionHeader n={4} title="Approach" />

<KthSmallestBSTDiagram />

Push all left-spine nodes onto a stack. Pop the top, increment a counter; when the counter equals `k`, return the popped node's value. Otherwise, move to the popped node's right child and repeat — pushing its left spine.

<Callout type="cyan" label="Why iterative over recursive">
Recursive inorder carries the entire call stack until it returns. The iterative version can return the moment <code>n == k</code> without unwinding — cleaner early exit and potentially lower peak stack depth.
</Callout>

<SectionHeader n={6} title="Step-by-Step Solution" />

<StepList>
  <StepItem title="Initialise stack and curr">
    `var stack: [TreeNode?] = []`, `var curr: TreeNode? = root`, `var n = 0`.
  </StepItem>
  <StepItem title="Push left spine">
    While `curr != nil`: push `curr` onto the stack, move `curr = curr?.left`.
  </StepItem>
  <StepItem title="Pop and count">
    Pop the top node, increment `n`. If `n == k`, return the node's value.
  </StepItem>
  <StepItem title="Move to right subtree">
    Set `curr = poppedNode?.right` and repeat the left-spine push from there.
  </StepItem>
</StepList>

<SectionHeader n={7} title="Code Implementation" />

```swift
class Solution {
  func kthSmallest(_ root: TreeNode?, _ k: Int) -> Int {
    var stack: [TreeNode?] = []
    var curr: TreeNode? = root
    var n = 0

    while !stack.isEmpty || curr != nil {
      while curr != nil {
        stack.append(curr)
        curr = curr?.left
      }
      curr = stack.removeLast()
      n += 1
      if n == k { return curr!.val }
      curr = curr?.right
    }

    return -1  // unreachable given valid k
  }
}
```
