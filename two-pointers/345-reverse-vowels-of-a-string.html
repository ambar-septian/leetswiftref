Reverse Vowels of a String

Two PointersString
01

Problem Overview

Given a string s, reverse only all the vowels in the string and return it.

The vowels are 'a', 'e', 'i', 'o', 'u', and they can appear in both lower and upper cases.

Constraints:

  • 1 <= s.length <= 3 * 10^5
  • s consists of printable ASCII characters.
02

Examples

Example 1​

Input:  s = "IceCreAm"
Output: "AceCreIm"
Explanation: The vowels are I, e, e, A. Reversed: A, e, e, I.

Example 2​

Input:  s = "leetcode"
Output: "leotcede"

04

Approach

Convert s to a [Character] array. left starts at 0, right at the last index. Advance left right while it points to a consonant; advance right left while it points to a consonant. When both point to vowels, swap and move both inward. Repeat until left >= right.

Four cases per iteration

Each iteration handles one of: (1) both are vowels — swap and advance both; (2) only left is a vowel — advance right; (3) only right is a vowel — advance left; (4) neither is a vowel — advance both. All cases converge correctly.

Step-by-Step Solution

Convert to character array —

var chars = s.map { $0 }. Define let vowels: Set = ["a","A","e","E","i","I","o","O","u","U"].

Initialise two pointers —

var left = 0, var right = chars.count - 1.

Advance past consonants and swap vowels —

While left less than right: handle all four vowel/consonant combinations — swap when both are vowels.

Return result —

return String(chars).

07

Code Implementation

class Solution {
  func reverseVowels(_ s: String) -> String {
    var chars  = s.map { $0 }
    var left   = 0
    var right  = chars.count - 1
    let vowels: Set<Character> = ["a","A","e","E","i","I","o","O","u","U"]

    while left < right {
      let l = chars[left], r = chars[right]
      if vowels.contains(l), vowels.contains(r) {
        chars.swapAt(left, right)
        left += 1; right -= 1
      } else if vowels.contains(l) {
        right -= 1
      } else if vowels.contains(r) {
        left += 1
      } else {
        left += 1; right -= 1
      }
    }

    return String(chars)
  }
}

09

Common Pitfalls

✕01
Missing uppercase vowels in the set
If only lowercase vowels are included, uppercase vowels like 'A' or 'I' are treated as consonants and not swapped.
Fix: Include all 10 vowels in the set: ["a","A","e","E","i","I","o","O","u","U"].
✕02
Advancing only one pointer when neither is a vowel
If only left or only right advances when both are consonants, the loop makes no progress when two consecutive consonants meet in the middle.
Fix: When neither is a vowel, advance both: left += 1; right -= 1.
✕03
Modifying the original string directly
Swift strings are value types and not index-subscriptable like arrays. Directly modifying String indices requires String.Index arithmetic which is error-prone.
Fix: Convert to [Character] first with s.map { $0 }, swap in-place, then convert back with String(chars).
