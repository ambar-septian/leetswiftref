Special Array I

Array
01

Problem Overview

An array is considered special if every pair of its adjacent elements contains two numbers with different parity (one even and one odd).

You are given an array of integers nums. Return true if nums is special, otherwise false.

Constraints:

  • 1 <= nums.length <= 100
  • 1 <= nums[i] <= 100
02

Examples

Example 1​

Input:  nums = [1]
Output: true
Explanation: A single-element array has no adjacent pairs — trivially special.

Example 2​

Input:  nums = [2,1,4]
Output: true
Explanation:
  (2,1): even-odd — different parity ✓
  (1,4): odd-even — different parity ✓
  All adjacent pairs alternate — special.

Example 3​

Input:  nums = [4,3,1,6]
Output: false
Explanation:
  (4,3): even-odd — different parity ✓
  (3,1): odd-odd — same parity ✗
  Not special.
03

Pattern Recognition

→ Pattern Decision Tree

Do we check every adjacent pair for opposite parity?
✓ Yes — A single pass comparing nums[i] % 2 with nums[i-1] % 2 suffices. Return false immediately on a match.
Tap to reveal pattern →
04

Approach

Iterate from index 1. For each pair (nums[i-1], nums[i]), check if they have the same parity: nums[i-1] % 2 == nums[i] % 2. If so, return false. If the loop completes, return true.

Parity shortcut

XOR of the two numbers: (nums[i-1] ^ nums[i]) % 2 == 0 means same parity. Alternatively, (nums[i-1] + nums[i]) % 2 == 0 also detects matching parity since even+even and odd+odd are both even.

06

Step-by-Step Solution

Handle edge case —

If nums.count == 1, return true immediately.

Check adjacent pairs —

For i in 1..<nums.count: if nums[i] % 2 == nums[i-1] % 2, return false.

Return true —

All pairs have different parity — return true.

07

Code Implementation

class Solution {
  func isArraySpecial(_ nums: [Int]) -> Bool {
    for i in 1..<nums.count {
      if nums[i] % 2 == nums[i - 1] % 2 { return false }
    }
    return true
  }
}

Common Pitfalls

✕01
Checking if both are even OR both are odd separately
Two conditions can be unified: if (a % 2 == b % 2), they have the same parity regardless of whether both are even or both are odd.
Fix: Use a single condition: nums[i] % 2 == nums[i-1] % 2.
