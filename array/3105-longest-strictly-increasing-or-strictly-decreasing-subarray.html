Longest Strictly Increasing or Strictly Decreasing Subarray

Array
01

Problem Overview

You are given an array of integers nums.

Return the length of the longest subarray of nums which is either strictly increasing or strictly decreasing.

Constraints:

  • 1 <= nums.length <= 50
  • 1 <= nums[i] <= 50
02

Examples

Example 1​

Input:  nums = [1,4,3,3,2]
Output: 2
Explanation:
  Strictly increasing subarrays: [1,4] (length 2).
  Strictly decreasing subarrays: [4,3], [3,2] (length 2).
  Maximum is 2.

Example 2​

Input:  nums = [3,3,3,3]
Output: 1
Explanation: All elements equal — no adjacent pair is strictly increasing or decreasing.

Example 3​

Input:  nums = [3,2,1]
Output: 3
Explanation: The whole array is strictly decreasing.
03

Pattern Recognition

→ Pattern Decision Tree
?
Can we track increasing and decreasing streaks independently?
✓ Yes — Maintain two counters: currentInc and currentDec. Extend the relevant one when the direction holds; reset the other.
?
What happens on a tie (equal elements)?
✓ Yes — Equal adjacent elements break both strictly increasing and strictly decreasing — reset both counters to 1.
Tap to reveal pattern →
04

Approach

Initialise currentInc = 1, currentDec = 1, maxInc = 1, maxDec = 1. Iterate from index 1. For each pair (prev, curr):

  • If curr > prev: extend currentInc, reset currentDec = 1.
  • If curr < prev: extend currentDec, reset currentInc = 1.
  • If curr == prev: reset both to 1.

Update the running maxima. Return max(maxInc, maxDec).

Step-by-Step Solution

Initialise counters —

var currentInc = 1, currentDec = 1, maxInc = 1, maxDec = 1.

Iterate from index 1 —

For each i in 1..<nums.count: compare nums[i] with nums[i-1].

  • Greater: currentInc += 1, maxInc = max(maxInc, currentInc), currentDec = 1.
  • Less: currentDec += 1, maxDec = max(maxDec, currentDec), currentInc = 1.
  • Equal: currentInc = 1, currentDec = 1.
Return max —

Return max(maxInc, maxDec).

07

Code Implementation

class Solution {
  func longestMonotonicSubarray(_ nums: [Int]) -> Int {
    var currentInc = 1, currentDec = 1
    var maxInc = 1, maxDec = 1

    for i in 1..<nums.count {
      let curr = nums[i], prev = nums[i - 1]
      if curr > prev {
        currentInc += 1
        maxInc = max(maxInc, currentInc)
        currentDec = 1
      } else if curr < prev {
        currentDec += 1
        maxDec = max(maxDec, currentDec)
        currentInc = 1
      } else {
        currentInc = 1
        currentDec = 1
      }
    }

    return max(maxInc, maxDec)
  }
}

09

Common Pitfalls

✕01
Not resetting the opposite counter
When curr > prev, the decreasing streak is broken. Failing to reset currentDec to 1 inflates the decreasing streak incorrectly.
Fix: On every direction change, reset the opposite counter to 1.
✕02
Treating equal elements as increasing or decreasing
Equal adjacent elements satisfy neither strict condition. Including them in either streak would overcount.
Fix: Add an explicit else branch that resets both counters to 1 when curr == prev.
