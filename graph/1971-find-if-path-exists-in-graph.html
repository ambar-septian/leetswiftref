Find if Path Exists in Graph

Depth-First SearchBreadth-First SearchUnion-FindGraph Theory
01

Problem Overview

There is a bi-directional graph with n vertices, where each vertex is labeled from 0 to n - 1 (inclusive). The edges in the graph are represented as a 2D integer array edges, where each edges[i] = [u_i, v_i] denotes a bi-directional edge between vertex u_i and vertex v_i. Every vertex pair is connected by at most one edge, and no vertex has an edge to itself.

You want to determine if there is a valid path that exists from vertex source to vertex destination.

Given edges and the integers n, source, and destination, return true if there is a valid path from source to destination, or false otherwise.

Constraints:

  • 1 <= n <= 2 * 10^5
  • 0 <= edges.length <= 2 * 10^5
  • edges[i].length == 2
  • 0 <= u_i, v_i <= n - 1
  • u_i != v_i
  • 1 <= source, destination <= n - 1
  • There are no duplicate edges.
  • There are no self-edges.

04

Approach

Build an undirected adjacency list. DFS from source. At each step, if a neighbor equals destination, return true. Otherwise, mark it visited and recurse. If the DFS exhausts all reachable nodes without finding destination, return false.

Array vs HashSet for visited

Using visited = Array(repeating: false, count: n) is faster than a Set for large n — O(1) lookup with better cache performance. Only use a HashSet when n is unknown or very large.

06

Step-by-Step Solution

Early exit —

If source == destination, return true immediately.

Build adjacency list —

For each edge [u, v], add v to list[u] and u to list[v].

DFS from source —

For each unvisited neighbor: if it equals destination, return true. Otherwise mark visited and recurse.

Return false —

If DFS completes without reaching destination, return false.

07

Code Implementation

class Solution {
  func validPath(_ n: Int, _ edges: [[Int]], _ source: Int, _ destination: Int) -> Bool {
    var visited = Array(repeating: false, count: n)
    var list = Array(repeating: [Int](), count: n)
    guard source != destination else { return true }

    for edge in edges {
      list[edge[0]].append(edge[1])
      list[edge[1]].append(edge[0])
    }

    func dfs(_ curr: Int) -> Bool {
      for neighbour in list[curr] {
        if neighbour == destination { return true }
        if !visited[neighbour] {
          visited[neighbour] = true
          if dfs(neighbour) { return true }
        }
      }
      return false
    }

    return dfs(source)
  }
}

09

Common Pitfalls

✕01
Marking visited after checking destination
If you mark visited before checking whether the neighbor is the destination, you might accidentally mark the destination as visited and never return true from it.
Fix: Check if neighbour == destination first, return true immediately. Only mark visited and recurse if it is not the destination.
✕02
Not marking nodes visited before recursing
In an undirected graph, not marking visited causes infinite cycles between connected nodes.
Fix: Set visited[neighbour] = true before calling dfs(neighbour).
✕03
Forgetting source == destination edge case
If source equals destination, the path trivially exists. Without this guard, DFS from source will never match destination in the neighbor check.
Fix: Guard with source != destination else return true at the top.
